# DPI与IRR解读

## 概述
DPI（Distribution to Paid-In）和IRR（Internal Rate of Return）是衡量VC/PE基金业绩的两个最核心指标，分别从"现金回报"和"时间价值"两个维度评估基金表现。理解这两个指标的定义、计算方法、适用场景及局限性，是LP评估GP能力、GP展示业绩的基础技能。正确解读这些指标对于投资决策和业绩沟通至关重要。

## 核心内容

### 一、核心业绩指标定义

#### 1. DPI（Distribution to Paid-In）已分配收益倍数

**定义**：基金已分配给投资人的现金与投资人已缴入资本的比率

**计算公式**：
$$DPI = \frac{已分配现金 + 已分配证券价值}{已缴入资本}$$

**解读**：
- DPI < 1：投资人尚未收回全部本金
- DPI = 1：投资人刚好收回全部本金
- DPI > 1：投资人已获得正向现金回报
- DPI = 2.0 表示投资人每投入1元已收回2元

**特点**：
- 只计算已实现的现金分配
- 不受账面估值影响
- 是衡量基金"真实回报"的黄金标准
- 在基金早期（前3-5年）通常较低

#### 2. IRR（Internal Rate of Return）内部收益率

**定义**：使基金所有现金流（包括投入和分配）的净现值等于零的折现率

**计算公式**：
$$\sum_{t=0}^{n} \frac{CF_t}{(1+IRR)^t} = 0$$

其中：$CF_t$ = 第t期的现金流（投入为负，分配为正）

**解读**：
- IRR反映资金的年化回报率
- 考虑了资金的时间价值
- IRR = 20% 表示资金年化增长20%

**特点**：
- 考虑了时间价值
- 可包含未实现投资的账面价值
- 容易被短期高估值项目"美化"
- 对现金流时点高度敏感

#### 3. TVPI（Total Value to Paid-In）总价值倍数

**定义**：基金总价值（已分配 + 未分配）与已缴入资本的比率

**计算公式**：
$$TVPI = \frac{已分配 + 未分配价值}{已缴入资本} = DPI + RVPI$$

**解读**：
- TVPI反映基金的总回报水平（含账面价值）
- 在基金存续中期最具参考价值
- 最终TVPI应等于DPI（基金结束时所有投资均已退出）

#### 4. RVPI（Residual Value to Paid-In）未分配收益倍数

**定义**：基金未分配价值与已缴入资本的比率

**计算公式**：
$$RVPI = \frac{未分配价值}{已缴入资本} = TVPI - DPI$$

**解读**：
- RVPI反映账面未实现价值
- 随着基金成熟应逐渐降低
- RVPI越高，说明基金越"年轻"

### 二、指标间的关系

**基金生命周期中各指标的典型变化**

```
投资期（前3-4年）：
  DPI: 0.0-0.3x  ← 少量早期退出
  RVPI: 0.8-1.5x ← 大量账面增值
  TVPI: 1.0-1.8x ← 以账面增值为主
  IRR: 5-25%     ← 波动较大

退出期（第5-8年）：
  DPI: 0.5-1.5x  ← 退出逐步增加
  RVPI: 0.5-1.0x ← 账面价值降低
  TVPI: 1.5-2.5x ← 总回报逐步显现
  IRR: 15-30%    ← 相对稳定

基金后期（第8-10年）：
  DPI: 1.5-3.0x  ← 大部分已退出
  RVPI: 0.1-0.3x ← 仅剩少量未退出
  TVPI: 1.8-3.0x ← 接近最终回报
  IRR: 18-35%    ← 基本确定
```

**指标换算关系**

| 组合状态 | DPI | RVPI | TVPI | 解读 |
|----------|-----|------|------|------|
| 早期基金 | 0.2x | 1.3x | 1.5x | 账面回报尚可，现金回报不足 |
| 中期基金 | 1.0x | 1.0x | 2.0x | 已回本，账面还有1倍 |
| 成熟基金 | 2.5x | 0.2x | 2.7x | 大部分已退出，回报确定 |

### 三、各指标的适用场景与局限性

#### DPI的适用场景

**适用**：
- 评估基金的真实现金回报能力
- 比较不同vintage year基金的实际表现
- LP的现金流规划和配置决策

**局限**：
- 基金早期DPI普遍较低，难以判断
- 不适合评估仍在投资期的基金
- 无法反映基金的增值潜力

#### IRR的适用场景

**适用**：
- 评估资金的使用效率
- 比较不同期限投资的回报水平
- 与其他资产类别的回报对比

**局限**：
- 容易被短期高估值"美化"
- 对现金流时点高度敏感
- 可能不代表可实现的回报水平
- 早期高IRR可能无法持续

**IRR"陷阱"案例**

| 场景 | 投资时间 | 投资金额 | 退出金额 | 退出时间 | IRR |
|------|----------|----------|----------|----------|-----|
| 快进快出 | 第1年 | 1000万 | 1500万 | 第2年 | 50% |
| 长期持有 | 第1年 | 1000万 | 3000万 | 第5年 | 25% |
| 延迟退出 | 第1年 | 1000万 | 5000万 | 第10年 | 17% |

虽然场景A的IRR最高（50%），但场景C的实际回报倍数最高（5x）

#### TVPI的适用场景

**适用**：
- 评估基金的总体回报水平
- 基金中期的业绩跟踪
- 同期基金的横向比较

**局限**：
- 包含未实现的账面价值
- 受估值方法和时机影响大
- 可能高估实际可实现的回报

### 四、J-Curve效应

**定义**：VC/PE基金在投资初期通常表现为负回报，随后逐渐转正，形成"J"形曲线

**J-Curve的形成原因**

1. **费用消耗**：管理费、设立费用在早期大量支出
2. **投资期现金流**：持续投入资本但尚无退出
3. **估值滞后**：新投资通常以成本价入账，未体现增值
4. **退出时序**：优质项目通常需要更长时间才能退出

**J-Curve的时间特征**

| 基金年份 | 典型净IRR | 原因 |
|----------|-----------|------|
| 第1年 | -20% ~ -30% | 设立费用、管理费 |
| 第2年 | -15% ~ -20% | 持续投资、少量减值 |
| 第3年 | -5% ~ 0% | 首批项目估值调整 |
| 第4年 | 0% ~ 10% | 部分项目增值显现 |
| 第5年 | 10% ~ 20% | 早期退出实现 |
| 第6-8年 | 15% ~ 30% | 大量退出、回报显现 |
| 第9-10年 | 18% ~ 35% | 最终回报确定 |

**缩短J-Curve的策略**
- 选择有快速退出渠道的投资（如并购标的）
- 适时进行基金重组（restructuring）
- 加速投后价值提升，缩短退出周期

### 六、业绩指标的综合应用

**评估基金的完整指标框架**

```
┌─────────────────────────────────────────────────────┐
│                    基金业绩评估框架                    │
├─────────────────────────────────────────────────────┤
│  回报指标        │  风险指标        │  效率指标        │
│  ├─ Net IRR      │  ├─ 最大回撤     │  ├─ 投资倍数     │
│  ├─ DPI          │  ├─ 回报波动率   │  ├─ 资本使用率   │
│  ├─ TVPI         │  ├─ 下行风险     │  ├─ 退出效率     │
│  └─ Gross IRR    │  └─ 失败率       │  └─ 费用比率     │
└─────────────────────────────────────────────────────┘
```

**多指标交叉验证**

| 基金表现 | IRR | DPI | TVPI | 解读 |
|----------|-----|-----|------|------|
| 优秀 | >25% | >2.0x | >2.5x | 全方位优秀 |
| 中等 | 15-25% | 1.0-2.0x | 1.5-2.5x | 中规中矩 |
| 需关注 | >30% | <0.5x | >2.0x | 高IRR但现金回报不足 |
| 风险信号 | <10% | >1.5x | <1.5x | 现金回报尚可但增值有限 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| DPI | Distribution to Paid-In | 已分配现金与已缴入资本的比率 |
| IRR | Internal Rate of Return | 内部收益率，使净现值为零的折现率 |
| TVPI | Total Value to Paid-In | 总价值（已分配+未分配）与已缴入资本的比率 |
| RVPI | Residual Value to Paid-In | 未分配价值与已缴入资本的比率 |
| J-Curve | J形曲线 | 基金早期负回报后逐渐转正的曲线形态 |
| Vintage Year | 基金成立年份 | 基金开始投资运作的年份 |
| Net IRR | 净内部收益率 | 扣除费用后的内部收益率 |
| Gross IRR | 毛内部收益率 | 未扣除费用的内部收益率 |

## 实务要点

- **优先关注DPI**：对于成熟基金（5年以上），DPI是评估真实回报的最重要指标
- **警惕高IRR陷阱**：早期基金的高IRR可能由少数高估值项目驱动，不代表可实现回报
- **结合vintage year评估**：不同市场环境成立的基金，业绩基准应有所调整
- **关注费用影响**：比较Net IRR和Gross IRR的差异，评估费用结构的合理性
- **建立长期跟踪机制**：定期更新基金业绩数据，观察J-Curve的演变

## 常见误区

- **过度依赖IRR**：IRR容易被短期高估值美化，尤其在基金早期可能产生误导
- **忽视DPI的重要性**：很多LP过分关注账面回报，忽视了现金回报才是最终目标
- **跨vintage year直接比较**：不同市场环境下成立的基金，简单比较指标没有意义
