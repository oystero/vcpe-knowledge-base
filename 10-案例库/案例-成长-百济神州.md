# 百济神州 — 中国创新药出海标杆，从C轮到三地上市

## 基本信息

| 项目 | 内容 |
|------|------|
| 公司全称 | 百济神州（BeiGene, Ltd.） |
| 成立时间 | 2010年 |
| 总部 | 北京/美国剑桥 |
| 创始人 | 王晓东、欧雷强（John Oyler） |
| 核心业务 | 创新型肿瘤药物的研发、生产和商业化 |
| 所属赛道 | 生物医药/创新药 |
| 当前状态 | 港交所（6160.HK）、上交所（688235）、纳斯达克（BGNE）三地上市 |

## 商业模式

百济神州采用"自主研发+全球化商业化+战略合作"的创新药商业模式：

1. **差异化管线布局**：核心产品BTK抑制剂泽布替尼（百悦泽）和PD-1单抗替雷利珠单抗（百泽安）均具备全球best-in-class潜力，而非简单的me-too药物。
2. **全球化临床和商业化**：在中国、美国、欧洲等地开展全球多中心临床试验，自主建设海外商业化团队，是中国biotech中全球化程度最高的企业之一。
3. **"不计成本"的研发投入**：早期年研发投入超过营收，高举高打的策略使其建立了业内最大的肿瘤免疫管线之一。
4. **战略合作补强**：与安进（Amgen）达成深度战略合作，引入地舒单抗等成熟产品，丰富产品组合。

## 投资逻辑（分轮次分析）

**C轮（2014年）**：高瓴资本领投，逻辑在于：（1）中国创新药政策环境即将迎来重大变革；（2）王晓东作为中国生命科学领域的顶级科学家，加上欧雷强的商业化能力，团队组合极具竞争力；（3）泽布替尼等核心管线具有全球竞争力。

**纳斯达克IPO（2016年）**：作为少数在美国上市的中国创新药企业，百济神州代表了中国biotech的国际化雄心。投资人看重的是其全球化管线布局和顶级管理团队。

**安进战略投资（2020年）**：安投入股约20.5亿美元，占股约20.5%，这是全球大型药企对中国biotech最大的单笔投资之一，验证了百济神州的全球竞争力。

## 关键转折点

- **2010年**：百济神州成立，由王晓东和欧雷强联合创办。
- **2014年**：高瓴资本领投C轮，公司进入发展快车道。
- **2016年**：纳斯达克上市，成为首批在美国上市的中国创新药企业。
- **2019年11月**：泽布替尼获得FDA批准，成为第一个在美获批的中国自主研发抗癌药。
- **2020年7月**：安进以约20.5亿美元战略入股，深度合作。
- **2021年12月**：A股科创板上市，募资221.6亿元，成为科创板医药类最大IPO之一。
- **2022-2023年**：泽布替尼全球销售额快速攀升，超过百亿美元峰值预期。
- **2024-2025年**：公司逐步实现盈利，从"烧钱研发"转向"盈利增长"阶段。

## 估值演变分析

| 轮次 | 时间 | 估值 | 估值方法 | 核心假设 |
|------|------|------|------|------|
| A轮 | 2011年 | 约5-10亿美元 | VC Method | 基于核心管线的潜在市场价值折现 |
| C轮 | 2014年 | 约15-25亿美元 | VC Method | 泽布替尼等管线的全球商业化潜力 |
| 纳斯达克IPO | 2016年 | 约50-70亿美元 | 可比公司法 | 参考全球biotech上市公司估值 |
| 港股IPO | 2018年 | 约100-150亿美元 | 可比公司法 | 管线进展+中国市场潜力 |
| 安进战略投资 | 2020年 | 约150亿美元 | 并购估值法 | 安进入股价格对应的估值水平 |
| A股IPO | 2021年 | 约2000亿元 | 可比公司法 | 科创板biotech估值溢价 |

**估值倍数变化趋势**：
- A轮到C轮：估值跳升3-5倍，管线进展推动
- C轮到纳斯达克IPO：估值增长2-3倍，上市流动性溢价
- 纳斯达克到港股：估值翻倍，中国市场价值重估
- 港股到A股：估值大幅跳升，科创板估值溢价显著

**估值逻辑演变**：早期采用VC Method（基于管线折现），上市后采用可比公司法（参考全球biotech估值），A股回归后享受科创板估值溢价。

1. **管线价值评估**：需验证核心管线的临床数据和商业化潜力，评估成功率
2. **团队能力验证**：需评估科学家+企业家组合的执行力和互补性
3. **全球化能力**：需验证公司在美国、欧洲等市场的临床和商业化能力
4. **资金使用效率**：需评估公司巨额研发投入的回报预期

## 投资人复盘

### 做对了什么
1. **押注顶级科学家+商业化人才的组合**：王晓东的科学判断力+欧雷强的商业运营能力是百济神州的核心竞争力。
2. **全球化战略的前瞻性**：在中国biotech普遍只做国内市场时，百济神州从一开始就定位全球化，这使其产品获得了更高的估值溢价。
3. **高举高打策略的正确性**：虽然早期巨额亏损引发争议，但大研发投入最终产出了全球级别的管线。
4. **三地上市的资本策略**：在不同市场上市，确保了充裕的资金支持长期研发。

### 可复制原则
- **投资创新药应重点关注"科学家+企业家"的团队组合**，纯科学家创业往往缺乏商业化能力。
- **全球化管线是中国创新药获得高估值的关键**：仅有国内市场的me-too药物估值空间有限。
- **耐心资本在创新药投资中至关重要**：百济神州从成立到盈利用了约15年，短期思维无法参与这类投资。
