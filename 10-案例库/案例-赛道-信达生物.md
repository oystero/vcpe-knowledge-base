# 信达生物 — PD-1先行者，中国创新药国际合作的标杆

## 基本信息

| 项目 | 详情 |
|------|------|
| 公司全称 | 信达生物制药（Innovent Biologics, Inc.） |
| 成立时间 | 2011年 |
| 总部 | 中国江苏苏州 |
| 创始人 | 俞德超 |
| 上市时间 | 2018年10月31日（港交所：1801） |
| IPO价格 | 13.98港元/股 |
| IPO募资 | 约33亿港元 |
| 2025年产品收入 | 约119亿元人民币 |
| 主营业务 | 创新生物药研发、生产和销售（肿瘤、自身免疫、代谢等） |

## 商业模式与竞争壁垒

信达生物采用"自主研发+国际合作"的商业模式，是中国创新药国际合作的标杆。

核心壁垒：
- **全面集成平台**：集研发、临床开发、生产制造、商业化于一体
- **国际合作**：与礼来7次合作，总交易额超100亿美元
- **GMP标准**：同时符合中国NMPA、美国FDA、欧盟EMA标准
- **丰富管线**：13款肿瘤产品，覆盖多疾病领域

## 行业地位与竞争格局

信达生物是中国创新药头部企业，PD-1领域的重要玩家。竞争对手包括：
- **恒瑞医药**：中国创新药"一哥"，PD-1市场份额最大
- **百济神州**：在PD-1和血液肿瘤领域竞争
- **君实生物**：PD-1领域的竞争对手
- **跨国药企**：默沙东、BMS等在中国市场的竞争

## 投资逻辑

**赛道逻辑**：PD-1/PD-L1是肿瘤免疫治疗的核心靶点，全球年销售额超300亿美元。中国创新药市场正处于爆发期，国际化是中国创新药企的必由之路。

**公司逻辑**：俞德超是中国生物医药领域最具国际视野的企业家之一，与礼来的7次合作证明了信达的国际竞争力。公司从PD-1起家，逐步扩展到多疾病领域。

## 关键转折点

1. **2011年**：俞德超创立信达生物
2. **2012年**：果断押注PD-1赛道
3. **2015年**：与礼来达成两次战略合作，总金额超15亿美元
4. **2018年**：港交所上市，成为江苏省首家港股生物医药企业
5. **2018年12月**：信迪利单抗获批上市
6. **2019年**：信迪利单抗进入国家医保目录
7. **2025年**：产品收入突破百亿，扭亏为盈

## 投资回报分析

- **D轮到IPO**：2.62亿美元到约33亿港元（约4亿美元），回报约**1.5倍**
- **IPO到2024年高点**：13.98港元到约109港元，回报约**8倍**
- **2025年**：股价回落至约90港元，总市值1556亿港元
- **创始人俞德超**：2025年身价130亿元

信达生物的投资回报体现了"创新药长周期"的特点——上市后长期亏损，直到2025年才扭亏为盈。

## 对该赛道投资的启示

1. **国际合作是创新药企的价值放大器**：信达与礼来的合作提升了公司的国际信誉
2. **PD-1是"百亿美元分子"**：但竞争激烈，需要差异化策略
3. **创新药投资需要极大的耐心**：信达上市后亏损6年才扭亏
4. **创始人国际视野很重要**：俞德超的美国背景帮助信达建立了国际合作
5. **苏州是中国创新药的重镇**：信达、基石等创新药企聚集在苏州
