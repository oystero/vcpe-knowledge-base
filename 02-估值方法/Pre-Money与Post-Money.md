# Pre-Money 与 Post-Money 估值

## 概述
Pre-Money（投资前估值）和 Post-Money（投资后估值）是 VC/PE 融资中最基础也最关键的估值概念，它们定义了投资人在每一轮融资中获得的股权比例。理解两者的关系、期权池（Option Pool）对估值的稀释效应，以及"完全稀释基础"的计算方式，是避免估值陷阱、保护各方利益的基础。

## 核心内容

### 基本定义与关系

| 概念 | 定义 | 公式 |
|------|------|------|
| Pre-Money Valuation | 投资人注资前公司的估值 | = Post-Money - 投资金额 |
| Post-Money Valuation | 投资人注资后公司的估值 | = Pre-Money + 投资金额 |
| 投资人股权比例 | 投资金额 / Post-Money | = 投资额 / (Pre-Money + 投资额) |

**示例：**
- Pre-Money = 8000万元
- 投资金额 = 2000万元
- Post-Money = 1亿元
- 投资人股权比例 = 2000 / 10000 = **20%**

### 期权池（Option Pool）对估值的影响

**期权池 Shuffle（Option Pool Shuffle）** 是硅谷 VC 交易中的常见做法：
- VC 要求在投资前（Pre-Money 内）预留员工期权池
- 这意味着期权池的稀释由创始团队承担，而非投资人
- 实际上降低了创始人的有效 Pre-Money 估值

**示例对比：**

| 方案 | Pre-Money | 期权池 | 有效Pre-Money | 投资额 | Post-Money |
|------|-----------|--------|--------------|--------|------------|
| 无期权池 | 8000万 | 0 | 8000万 | 2000万 | 1亿 |
| 期权池在Pre-Money内 | 8000万 | 1000万（10%） | 7000万 | 2000万 | 1亿 |
| 期权池在Post-Money内 | 8000万 | 1000万（10%） | 8000万 | 2000万 | 1.1亿 |

**关键差异分析（期权池在 Pre-Money 内）：**
- 名义 Pre-Money = 8000万
- 但期权池 1000万 从 Pre-Money 中扣除
- 创始人实际持股基础：8000万 - 1000万 = 7000万
- 创始人有效估值被降低

### 完全稀释基础（Fully Diluted Basis）

**完全稀释 = 已发行股份 + 期权池预留 + 可转换证券 + 其他潜在稀释**

| 稀释来源 | 说明 | 示例 |
|----------|------|------|
| 已发行普通股 | 创始人、早期员工持有的股份 | 1000万股 |
| 已授出期权 | 已授予但未行权的员工期权 | 200万股 |
| 未授出期权池 | 预留但尚未授予的期权 | 300万股 |
| 可转换债券（SAFE/可转债） | 在本轮融资中将转换为股权的债务 | 150万股等值 |
| 认股权证（Warrant） | 历史交易中附带的认股权 | 50万股 |
| **完全稀释总股数** | | **1700万股** |

**实务要点：**
- VC 几乎总是要求按完全稀释基础计算股权比例
- 创始人应关注"期权池是否在 Pre-Money 内"——这是谈判的关键点
- SAFE/可转债的转换折扣和上限会影响实际稀释

### 估值陷阱与常见博弈

**陷阱 1：名义高估值 + 大期权池**
- Pre-Money 1亿 + 期权池 20% = 创始人实际获得的估值只有 8000万水平
- VC 对外宣称"给了1亿估值"，实际成本更低

**陷阱 2：忽略 SAFE/可转债的稀释**
- 创业公司在正式 A 轮前可能发行了多笔 SAFE 或可转债
- 这些在 A 轮自动转换，会显著稀释创始人和 A 轮投资人的股权
- 计算时必须纳入

**陷阱 3：不区分 Pre-Money 和 Post-Money 的表述**
- "公司估值1亿"——这到底是 Pre-Money 还是 Post-Money？
- 媒体报道常常混淆，务必在条款清单中明确

### 各轮次典型估值参考（2023-2025中国市场）

| 轮次 | Pre-Money 估值区间 | 投资金额区间 | 期权池 |
|------|-------------------|-------------|--------|
| 种子轮 | 500万-3000万 | 100万-500万 | 10%-15% |
| Pre-A | 2000万-8000万 | 500万-2000万 | 10%-15% |
| A轮 | 5000万-3亿 | 1000万-5000万 | 10%-20% |
| B轮 | 3亿-15亿 | 5000万-2亿 | 5%-10% |
| C轮 | 10亿-50亿+ | 1亿-5亿 | 5%-10% |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 投资前估值 | Pre-Money Valuation | 投资人资金注入前公司的估值 |
| 投资后估值 | Post-Money Valuation | 投资人资金注入后公司的估值（= Pre-Money + 投资额） |
| 期权池 | Option Pool / ESOP | 为吸引和激励员工预留的股权池 |
| 期权池倒挤 | Option Pool Shuffle | 在 Pre-Money 中扣除期权池，由创始人承担稀释 |
| 完全稀释 | Fully Diluted | 假设所有可转换证券、期权均已行权后的总股本 |
| SAFE | Simple Agreement for Future Equity | Y Combinator 创建的未来股权简单协议 |
| 可转换债券 | Convertible Note | 在触发条件下可转换为股权的债务工具 |

## 实务要点
- **谈判时明确期权池位置**：争取让期权池在 Post-Money 中设置（而非 Pre-Money），或至少缩小池子比例
- **计算 SAFE 的实际稀释**：多笔 SAFE 的转换需考虑各自不同的折扣率（Discount）和估值上限（Valuation Cap），用模拟表逐一计算
- **关注条款清单中的定义**：条款清单中 "on a fully diluted basis" 是否包含期权池预留，需逐字确认
- **预留足够的期权池**：早期公司通常需要 10%-20% 的期权池来吸引核心人才，过小会导致后续融资时不得不扩大池子
- **使用 Cap Table 工具管理**：使用 Carta、LTSE 或 Excel Cap Table 模型管理股权结构，避免手工计算出错

## 常见误区
- **只关注 Pre-Money 忽略 Post-Money**：创始人听到"1亿估值"很兴奋，但加上 20% 期权池和 20% 投资额，创始人持股可能只剩 60%
- **将期权池视为"免费的"**：期权池来自 Pre-Money，本质是创始团队的让步，不是投资人的贡献
- **忽略可转债的累计稀释**：创业初期发了多笔 SAFE 和可转债而不记录，到 A 轮时才发现被大幅稀释
