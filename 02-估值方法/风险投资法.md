# 风险投资法（VC Method）

## 概述
风险投资法（Venture Capital Method，简称 VC Method）是一种从预期退出价值倒推当前投资估值的方法，由哈佛商学院教授 William Sahlman 于 1987 年提出。它是最适合早期风险投资的估值方法，核心逻辑是：先估算退出时的企业价值，再用目标 IRR 折现回来，从而确定当前可接受的投资价格。

## 核心内容

### 基本公式

**Step 1：估算退出价值（Exit Value）**
```
退出价值 = 退出年份的财务指标 x 退出倍数
```
常用指标：净利润、EBITDA、营收、ARR

**Step 2：计算当前投资后估值（Post-Money Valuation）**
```
投资后估值 = 退出价值 / (1 + 目标IRR)^n
```
其中 n 为投资到退出的年数（通常 3-7 年）

**Step 3：计算投资价格**
```
投资金额 = 投资后估值 × 所占股权比例
```

### IRR 目标设定（阶段差异）

| 投资阶段 | 目标IRR | 预期持有期 | 预期回报倍数 | 典型投资规模 |
|----------|---------|-----------|-------------|-------------|
| 天使轮（Angel） | 50%-70% | 5-8年 | 10x-30x | 50万-500万 |
| 种子轮（Seed） | 50%-60% | 5-7年 | 8x-20x | 100万-1000万 |
| A轮（Series A） | 40%-50% | 4-6年 | 5x-10x | 500万-3000万 |
| B轮（Series B） | 30%-40% | 3-5年 | 3x-6x | 2000万-1亿 |
| C轮及以后 | 25%-35% | 2-4年 | 2x-4x | 1亿以上 |
| Pre-IPO/PE | 20%-30% | 1-3年 | 1.5x-3x | 视项目而定 |

**IRR 与回报倍数的关系：**
- IRR 30%，持有 5 年 → 约 3.7x 回报
- IRR 40%，持有 5 年 → 约 5.4x 回报
- IRR 50%，持有 5 年 → 约 7.6x 回报

### 完整计算示例

**场景：某 SaaS 公司 A 轮融资**

| 参数 | 数值 |
|------|------|
| 当前年营收 | 1000万元 |
| 预计第5年营收 | 1亿元 |
| 退出时 EV/Revenue 倍数 | 8x |
| 退出估值 | 8亿元 |
| VC 目标 IRR | 40% |
| 投资后估值 | 8亿 / (1.4)^5 = 1.50亿元 |
| 投资金额 | 5000万元 |
| 所占股权比例 | 5000万 / 1.50亿 = 33.3% |

### VC Method 的扩展应用

**1. 多情景加权法**
| 情景 | 概率 | 退出估值 | 加权退出估值 |
|------|------|----------|-------------|
| 大成（IPO） | 15% | 30亿 | 4.5亿 |
| 中成（并购） | 35% | 8亿 | 2.8亿 |
| 小成（次轮退出） | 30% | 3亿 | 0.9亿 |
| 失败 | 20% | 0 | 0 |
| **期望退出估值** | | | **8.2亿** |

**2. 风险调整后的概率方法**
- 将失败概率纳入 IRR：要求更高的 IRR 来补偿下行风险
- 部分 VC 用 "required hit rate" 来评估：如果 10 个项目只有 1 个成功，那成功项目需要 10x+ 的回报

### VC Method 的局限性

| 局限 | 说明 |
|------|------|
| 高度依赖退出假设 | 退出倍数、退出时间的微小变化会导致估值大幅波动 |
| 忽略中间融资稀释 | 后续轮次融资会稀释当前投资人的股权，需额外建模 |
| 不考虑现金流 | 与 DCF 不同，不分析企业的实际现金创造能力 |
| IRR 主观性强 | 目标 IRR 因 GP/LP 而异，缺乏客观标准 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 内部收益率 | Internal Rate of Return (IRR) | 使投资净现值为零的折现率，衡量投资回报的核心指标 |
| 退出价值 | Exit Value / Terminal Value | 投资退出时企业或股权的预期价值 |
| 投资倍数 | Multiple on Invested Capital (MOIC) | 退出价值 / 投入资本，不考虑时间价值 |
| 目标回报率 | Target Return / Hurdle Rate | VC 要求的最低投资回报率 |
| 稀释效应 | Dilution Effect | 后续融资轮次对现有股东持股比例的摊薄 |
| 退出路径 | Exit Path | 投资退出的方式，包括 IPO、并购、二级市场转让、回购等 |

## 实务要点
- **IRR 目标应反映风险溢价**：不同行业、不同阶段的风险差异巨大，生物技术 A 轮的 IRR 目标应高于 SaaS B 轮
- **充分考虑稀释**：A 轮投资人在退出前通常经历 3-5 轮融资，每轮稀释 10%-20%，需在模型中预留空间
- **退出倍数要基于可比案例**：参考同行业近期的并购/IPO 案例，而非凭空假设
- **"拍脑袋"也需有逻辑**：VC Method 本身带有一定的主观性，但每个假设都应有行业数据或案例支撑
- **结合其他方法交叉验证**：特别是当 VC Method 估值与可比公司法差距较大时，需深入分析原因

## 常见误区
- **忽略稀释导致估值偏高**：不考虑后续轮次稀释，导致当前轮次期望回报实际无法实现
- **退出倍数过于乐观**：牛市中容易高估退出倍数，应参考完整周期的平均倍数
- **混淆 IRR 与 MOIC**：IRR 考虑时间价值，MOIC 不考虑；5 年 3x（IRR~25%）与 2 年 3x（IRR~73%）差异巨大
