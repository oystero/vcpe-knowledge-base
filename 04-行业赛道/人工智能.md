# 人工智能

## 概述
人工智能（AI）是当前全球科技投资最热门的赛道，以GPT系列为代表的大语言模型（LLM）开启了AI的新纪元。AI产业链涵盖算力基础设施、基础模型、中间层工具和应用层。中国AI产业在应用创新方面具有优势，但在基础模型和高端算力方面面临"卡脖子"挑战。2023-2025年是AI投资的高峰期，大量资本涌入大模型和AI应用赛道。

## 行业概况
- **全球市场规模**：2025年全球AI市场规模约5000亿美元，CAGR超30%
- **中国市场规模**：2025年中国AI产业规模约8000亿人民币，AI大模型市场约500亿
- **产业链结构**：
  - **基础设施层**：AI芯片（GPU/TPU）、服务器、数据中心、云计算（英伟达、华为昇腾、寒武纪）
  - **模型层**：基础大模型（OpenAI、智谱AI、百度文心、阿里通义、字节豆包）
  - **中间层**：模型微调、RAG、向量数据库、AI Agent框架（LangChain、Dify）
  - **应用层**：AI助手、AI编程、AI营销、AI医疗、AI金融等垂直应用
- **主要玩家**：
  - 大模型：智谱AI、月之暗面（Kimi）、百川智能、MiniMax、零一万物
  - AI应用：百度（文心一言）、阿里（通义千问）、字节（豆包）
  - AI芯片：寒武纪、海光信息、壁仞科技
  - AI基础设施：商汤科技、第四范式、云从科技

## 投资逻辑
- **为什么值得投**：
  - AI是继互联网、移动互联网之后的第三波技术革命
  - 大模型能力快速提升，AI应用场景持续扩展
  - 中国拥有丰富的应用场景和数据优势
  - 企业级AI应用（如AI客服、AI编程、AI营销）已开始商业化
- **核心驱动力**：
  - 大模型能力的指数级提升（Scaling Law）
  - 算力成本持续下降
  - 企业降本增效的强烈需求
  - 政策支持（新一代人工智能发展规划）
- **关键成功因素**：
  - 模型能力（参数规模、训练数据质量、推理效率）
  - 场景理解与产品化能力
  - 算力资源获取能力
  - 商业化变现能力

## 估值特点
- **常用估值方法**：
  - **可比公司法**：参考OpenAI、Anthropic等全球头部AI公司的估值
  - **PS（Price/Sales）**：适用于已有收入但未盈利的AI公司
  - **用户价值法**：单用户价值 x 用户数
  - **技术资产估值**：模型参数量、训练数据量、算力储备
- **关键指标**：
  - **模型参数量**：衡量模型规模的基础指标
  - **Benchmark表现**：MMLU、HumanEval、SuperCLUE等评测分数
  - **API调用量/MAU**：应用层的用户活跃度
  - **ARR**：企业级AI应用的经常性收入
  - **推理成本**：每百万token的计算成本
- **估值区间参考**：
  - 大模型公司（Pre-revenue）：10-100亿美元（参考OpenAI的3000亿美元估值）
  - AI应用公司：PS 15-40x
  - AI芯片公司：PS 20-50x
  - AI基础设施：PS 10-25x

## 尽调要点
- **该行业特有的尽调关注点**：
  - 模型技术实力：自研能力 vs 调用开源模型
  - 算力资源：GPU储备、训练成本、推理成本
  - 数据优势：独有数据源、数据飞轮效应
  - 商业化路径：To B vs To C，付费意愿验证
  - 团队背景：核心技术人员的学术与产业背景
  - 合规风险：内容安全、数据安全、算法备案
- **常见红旗信号**：
  - 过度依赖开源模型，缺乏核心自研能力
  - 模型能力无明显差异化
  - 商业化路径不清晰，长期烧钱无造血能力
  - 核心技术人员流失
  - 合规风险（内容安全、数据安全）

## 典型案例
- **商汤科技（0020.HK）**：
  - 中国AI视觉龙头，2021年港股上市
  - 上市后持续亏损，2022年创始人汤晓鸥去世，公司战略调整
  - 2023年后转向大模型与AI基础设施（大装置）
  - 投资启示：AI公司商业化是最大挑战，技术领先不等于商业成功
- **智谱AI**：
  - 中国大模型创业公司代表，清华系背景
  - 2024年估值超100亿人民币，获得多轮大额融资
  - 产品包括ChatGLM系列模型、智谱清言应用
  - 投资启示：大模型创业需要持续巨额资金支持，赢者通吃的行业特征明显

## 风险与挑战
- **行业特有风险**：
  - 技术迭代风险：模型能力可能快速被超越
  - 商业化不确定性：AI应用的付费意愿和商业模式仍在探索
  - 人才竞争：核心AI人才稀缺，薪资成本高
  - 开源模型冲击：开源模型能力提升可能削弱闭源模型的壁垒
- **政策风险**：
  - 算力出口管制：美国对华GPU出口限制影响训练能力
  - 内容安全监管：生成式AI内容管理办法
  - 算法备案与合规要求
- **技术迭代风险**：
  - Scaling Law可能遇到瓶颈
  - 新架构（如Transformer替代方案）可能出现
  - 多模态、Agent等新范式的发展不确定性

## 关键术语
| 术语 | 英文 | 定义 |
|------|------|------|
| 大语言模型 | LLM (Large Language Model) | 参数量达数十亿至万亿级的语言模型，如GPT、Claude |
| 提示工程 | Prompt Engineering | 设计输入提示以引导大模型输出期望结果的技术 |
| RAG | Retrieval-Augmented Generation | 检索增强生成，结合外部知识库提升模型回答准确性 |
| AI Agent | 智能体 | 能够自主规划、使用工具、完成复杂任务的AI系统 |
| Scaling Law | 规模定律 | 模型性能随参数量、数据量、算力增加而持续提升的规律 |
| Token | 词元 | 大模型处理文本的基本单位，中文约1.5-2个字符为一个token |
| Fine-tuning | 微调 | 在预训练模型基础上用特定数据进行二次训练 |
| 推理 | Inference | 模型训练完成后，使用模型进行预测/生成的过程 |
| GPU | Graphics Processing Unit | 图形处理器，当前AI训练和推理的核心算力硬件 |
