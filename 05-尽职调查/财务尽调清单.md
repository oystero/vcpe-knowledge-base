# 财务尽调清单

## 概述
财务尽职调查（Financial Due Diligence）是对目标公司财务状况、经营成果和现金流量进行系统性审查的过程。它是投资决策的核心环节，直接关系到估值合理性和交易风险判断。财务尽调不仅验证历史数据的真实性，更要识别潜在的财务风险和未来盈利的可持续性。

## 核心内容

### 一、财务报表审查

#### 1. 三大报表分析

| 报表 | 核心审查点 | 关注指标 |
|------|-----------|---------|
| 资产负债表 | 资产质量、负债完整性、所有者权益真实性 | 资产负债率、流动比率、速动比率 |
| 利润表 | 收入确认合规性、成本费用合理性、盈利质量 | 毛利率、净利率、EBITDA |
| 现金流量表 | 经营性现金流与利润匹配度、资本支出合理性 | 经营现金流/净利润、自由现金流 |

#### 2. 审查 Checklist

- [ ] 获取最近3-5年经审计的财务报表及审计报告
- [ ] 审查审计意见类型（无保留/保留/否定/无法表示意见）
- [ ] 核对审计报告中的关键审计事项（Key Audit Matters）
- [ ] 检查会计政策变更及追溯调整影响
- [ ] 验证期后事项披露的完整性
- [ ] 对比管理报表与审计报表的差异
- [ ] 审查合并报表范围及变动原因

### 二、收入确认审查

#### 收入真实性验证

- [ ] 核查前十大客户的合同、发票、银行流水是否匹配
- [ ] 验证收入的截止性测试（Cut-off Test）
- [ ] 检查是否存在期末突击确认收入的情况
- [ ] 审查退货、折让、折扣政策及实际执行情况
- [ ] 核实收入确认政策是否符合 ASC 606 / IFRS 15 / 新收入准则
- [ ] 对比行业平均收入增长率，识别异常波动
- [ ] 抽样检查收入对应的物流单据、签收单、验收报告

#### 收入质量分析

| 维度 | 优质信号 | 风险信号 |
|------|---------|---------|
| 客户集中度 | 前五大客户占比 < 30% | 前五大客户占比 > 60% |
| 收入可预测性 | 长期合同、订阅模式 | 一次性项目为主 |
| 收入增长驱动 | 量价齐升 | 靠降价拉动销量 |
| 应收账款周转 | 应收增速 < 收入增速 | 应收增速远超收入增速 |

### 三、关联交易审查

- [ ] 编制完整的关联方清单（包括实控人亲属、关联企业）
- [ ] 审查关联交易的定价公允性（Transfer Pricing）
- [ ] 核查关联交易的必要性和商业合理性
- [ ] 检查关联方资金往来及余额
- [ ] 验证关联交易的信息披露完整性
- [ ] 评估去关联交易的可行性及对财务的影响
- [ ] 关注体外循环（Off-balance Sheet）风险

### 四、税务合规审查

- [ ] 获取最近3-5年纳税申报表及完税证明
- [ ] 核查增值税、企业所得税、个人所得税的合规性
- [ ] 审查税收优惠政策的合法性和可持续性
- [ ] 检查是否存在税务稽查或补税风险
- [ ] 验证研发费用加计扣除的合规性
- [ ] 审查股权激励相关的税务处理
- [ ] 评估跨境交易的税务合规性（如有）

### 五、现金流验证

#### 现金流质量分析

| 指标 | 计算公式 | 健康标准 |
|------|---------|---------|
| 经营现金流/净利润 | 经营活动现金流净额 / 净利润 | > 1.0（持续为正） |
| 自由现金流 | 经营现金流 - 资本支出 | 持续为正 |
| 现金转换周期 | DSO + DIO - DPO | 行业合理范围内 |
| 收现比 | 销售收到的现金 / 营业收入 | > 1.0 |

- [ ] 核查银行对账单与账面现金余额的一致性
- [ ] 验证大额收支的银行流水
- [ ] 检查是否存在账外资金或小金库
- [ ] 审查现金流量表的编制准确性
- [ ] 关注经营现金流与净利润长期背离的情况

### 六、审计报告分析

#### 审计意见类型解读

| 意见类型 | 含义 | 投资影响 |
|---------|------|---------|
| 无保留意见 | 财务报表公允反映 | 正常推进 |
| 带强调事项段的无保留意见 | 整体公允但有需关注事项 | 需重点关注强调事项 |
| 保留意见 | 部分事项不符合准则 | 需深入调查保留事项 |
| 否定意见 | 财务报表不能公允反映 | 重大风险，审慎考虑 |
| 无法表示意见 | 审计范围受限 | 重大不确定性 |

- [ ] 检查关键审计事项（KAM）的具体内容
- [ ] 关注审计机构的声誉和独立性
- [ ] 审查管理层估计和判断的合理性
- [ ] 核查重大会计差错更正及原因

### 七、常见财务造假手法识别

#### 收入端造假

| 手法 | 识别方法 | 核查手段 |
|------|---------|---------|
| 虚构客户/合同 | 前十大客户工商信息核查 | 天眼查/企查查验证、实地走访 |
| 提前确认收入 | 期末收入异常飙升 | 截止性测试、期后退货检查 |
| 渠道压货（Channel Stuffing） | 库存大幅增加、退货率上升 | 经销商库存核查、退货政策分析 |
| 关联交易虚增收入 | 交易对手为关联方 | 关联方穿透、资金流向追踪 |
| 循环交易 | 买卖同一批货物 | 物流单据核查、毛利率异常分析 |

#### 利润端造假

| 手法 | 识别方法 | 核查手段 |
|------|---------|---------|
| 资本化费用 | 研发/利息资本化比例异常高 | 对比同行业资本化比例 |
| 少计成本费用 | 毛利率显著高于同行 | 成本结构分析、BOM核对 |
| 利用减值准备调节利润 | 资产减值"大洗澡"后转回 | 减值准备计提政策分析 |
| 利用关联交易转移利润 | 向关联方低价采购/高价销售 | 公允价值对比分析 |

#### 资产端造假

| 手法 | 识别方法 | 核查手段 |
|------|---------|---------|
| 虚增存货 | 存货周转率持续下降 | 实地盘点、第三方仓储核查 |
| 虚增固定资产 | 在建工程长期不转固 | 实地勘察、设备清单核对 |
| 隐藏应收账款坏账 | 应收账款账龄异常延长 | 账龄分析、函证确认 |
| 虚增银行存款 | 资金余额与银行对账单不符 | 独立发函、银行流水核查 |

### 八、财务预测与估值基础

- [ ] 审查管理层财务预测（Financial Model）的假设合理性
- [ ] 对比行业基准验证关键假设（增长率、毛利率、费用率）
- [ ] 进行敏感性分析（Sensitivity Analysis）
- [ ] 评估营运资本需求及变动趋势
- [ ] 核查或有负债和表外风险

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 尽职调查 | Due Diligence (DD) | 投资前对目标公司进行全面调查和评估的过程 |
| EBITDA | Earnings Before Interest, Taxes, Depreciation and Amortization | 息税折旧摊销前利润，衡量经营业绩的核心指标 |
| 收现比 | Cash Receipt Ratio | 销售收到的现金与营业收入的比率，反映收入的现金含量 |
| 关联交易 | Related Party Transaction | 关联方之间转移资源、劳务或义务的行为 |
| 截止性测试 | Cut-off Test | 验证收入/支出是否记录在正确会计期间的审计程序 |
| 关键审计事项 | Key Audit Matters (KAM) | 审计师认为在审计中最重要的事项 |
| 自由现金流 | Free Cash Flow (FCF) | 经营现金流减去资本支出，反映企业可自由支配的现金 |
| 资本化费用 | Capitalized Costs | 将本应费用化的支出计入资产，可能导致利润虚增 |
| 渠道压货 | Channel Stuffing | 通过向经销商大量铺货来虚增收入的手法 |
| 大洗澡 | Big Bath | 通过集中计提大额减值准备为未来利润释放做准备 |

## 实务要点

1. **交叉验证原则**：任何财务数据都应通过至少两个独立来源交叉验证。银行流水、合同、发票、物流单据应相互印证，单一来源的数据可信度有限。

2. **关注异常偏离**：将目标公司财务指标与同行业上市公司对标，显著偏离行业均值的指标需要重点解释。毛利率高于同行10个百分点以上需要特别警惕。

3. **重视现金流**：利润可以被操纵，但现金流更难造假。长期经营现金流低于净利润是危险信号。关注"经营活动现金流/净利润"比率是否持续健康。

4. **审计报告不是万能的**：即使获得无保留意见的审计报告，也不能完全排除财务风险。审计本身存在固有局限性，投资人应独立判断。

5. **管理层配合度**：尽调过程中管理层的配合程度本身就是重要信号。拖延提供资料、反复变更数据口径、拒绝提供原始凭证等行为需引起高度警惕。

## 常见误区

1. **过度依赖历史财务数据**：历史数据只能说明过去，不能预测未来。投资人容易陷入"历史业绩好 = 未来也会好"的思维定式，忽略了行业变化、竞争格局、技术迭代等外部因素对未来的潜在影响。

2. **忽视营运资本分析**：很多投资人关注利润表而忽视资产负债表中的营运资本变化。快速增长的公司可能因为应收账款和存货大幅增加而面临现金流危机，即"增长悖论"（Growth Paradox）。

3. **低估关联交易风险**：在关联交易占比不高的情况下，投资人容易放松审查。但关联交易即使金额不大，也可能反映公司治理缺陷，且可能通过复杂的关联网络隐藏真实利益输送。
