# 法律尽调清单

## 概述
法律尽职调查（Legal Due Diligence）是对目标公司的法律合规性、权属清晰性和潜在法律风险进行全面审查的过程。法律尽调的核心目标是识别可能导致投资损失、估值调整或交易结构变更的法律风险，确保投资标的的法律基础稳固可靠。

## 核心内容

### 一、公司设立与存续合法性

#### 审查 Checklist

- [ ] 核查营业执照、公司章程、历次章程修正案
- [ ] 验证公司设立程序的合法性（发起人协议、创立大会记录等）
- [ ] 检查历次增资、减资的股东会决议及工商变更登记
- [ ] 审查公司存续状态（是否被列入经营异常名录、严重违法失信名单）
- [ ] 核查经营范围与实际业务是否匹配
- [ ] 检查分支机构、子公司的设立及存续情况
- [ ] 验证外资准入合规性（如适用，参照外商投资负面清单）

#### 重点关注

| 事项 | 审查要点 | 风险等级 |
|------|---------|---------|
| 出资瑕疵 | 实缴资本是否到位、非货币出资是否评估 | 高 |
| 设立程序瑕疵 | 是否存在代持、抽逃出资 | 高 |
| 工商信息不一致 | 注册地址、经营范围与实际不符 | 中 |
| 外资准入限制 | 负面清单行业、VIE结构合规性 | 高 |

### 二、股权结构与股东权利

#### 审查 Checklist

- [ ] 编制完整的股权穿透图（直至最终自然人/国资主体）
- [ ] 核查代持协议及还原安排
- [ ] 审查股东协议（Shareholders' Agreement）中的特殊权利条款
- [ ] 检查反稀释条款（Anti-dilution）、优先清算权（Liquidation Preference）
- [ ] 验证领售权（Drag-along Right）、随售权（Tag-along Right）
- [ ] 审查优先认购权（Pre-emptive Right）、回购权（Redemption Right）
- [ ] 检查是否存在股东诉讼或股东间纠纷
- [ ] 核查股权质押、冻结情况

#### 股权架构分析

| 架构类型 | 关注要点 | 常见风险 |
|---------|---------|---------|
| 创始人代持 | 代持原因、代持协议效力、还原可行性 | 代持人道德风险、还原税费 |
| 员工持股平台 | 平台类型（有限合伙/资管计划）、退出机制 | GP控制权、税务处理 |
| VIE结构 | 协议控制效力、外汇登记、利润回流 | 政策风险、协议可执行性 |
| 一致行动人 | 协议范围、解除条件、稳定性 | 控制权变更风险 |
| 国有股权 | 国资审批程序、评估定价、进场交易 | 审批延迟、定价约束 |

### 三、知识产权

#### 审查 Checklist

- [ ] 编制知识产权清单（专利、商标、著作权、商业秘密、域名）
- [ ] 核查专利的有效性、年费缴纳情况、权利归属
- [ ] 审查专利是否存在被无效宣告的风险
- [ ] 验证核心技术人员的知识产权归属（职务发明/非职务发明）
- [ ] 检查员工竞业禁止协议和保密协议（NDA）的签署情况
- [ ] 审查开源软件（Open Source Software）的使用合规性
- [ ] 核查知识产权的许可和被许可情况
- [ ] 检查是否存在知识产权侵权诉讼或争议

#### 知识产权价值评估

| 维度 | 高价值信号 | 风险信号 |
|------|-----------|---------|
| 专利质量 | 核心专利覆盖关键技术路径 | 大量低质量实用新型/外观设计 |
| 专利布局 | 多国申请、布局合理 | 仅在国内申请、布局单一 |
| 商业秘密 | 完善的保密制度和技术壁垒 | 核心技术依赖个别人员 |
| 商标保护 | 核心品牌全类别注册 | 商标被抢注或存在争议 |

### 四、重大合同审查

#### 审查 Checklist

- [ ] 列出所有重大合同清单（金额超过阈值或业务关键合同）
- [ ] 审查客户合同中的价格条款、付款条件、违约责任
- [ ] 检查供应商合同的排他性条款、最低采购量承诺
- [ ] 核查租赁合同的期限、租金、续租条件
- [ ] 审查贷款及担保合同（抵押、质押、保证）
- [ ] 检查合同中的控制权变更条款（Change of Control）
- [ ] 审查竞业禁止协议的范围和合理性
- [ ] 核查框架协议的实际执行情况与合同约定的偏差

#### 合同风险等级分类

| 风险类型 | 具体表现 | 处理建议 |
|---------|---------|---------|
| 控制权变更触发 | CoC条款允许对方终止合同 | 交易前取得豁免或承诺函 |
| 排他性约束 | 限制与其他方合作 | 评估业务影响，必要时调整交易结构 |
| 过度承诺 | 不切实际的业绩保证或赔偿条款 | 量化潜在损失，纳入估值调整 |
| 知识产权风险 | 合同中IP归属不清晰 | 补签IP归属协议 |

### 五、劳动用工

#### 审查 Checklist

- [ ] 核查劳动合同签订率及合规性（试用期、合同期限）
- [ ] 审查社保和公积金缴纳合规性
- [ ] 检查股权激励计划（ESOP）的法律架构和税务处理
- [ ] 核查核心员工的竞业禁止和保密协议
- [ ] 审查劳动争议历史及潜在纠纷
- [ ] 检查外包/劳务派遣用工的合规性
- [ ] 核查加班管理及加班费支付合规性
- [ ] 审查员工手册和规章制度的民主程序

#### 劳动用工合规重点

| 事项 | 合规要求 | 常见违规 |
|------|---------|---------|
| 社保公积金 | 按实际工资基数足额缴纳 | 按最低基数缴纳 |
| 竞业限制 | 补偿金不低于月均工资30% | 未支付补偿金或补偿过低 |
| 劳务派遣 | 用工比例不超过10%、仅限临时性/辅助性岗位 | 超比例使用、同工不同酬 |
| 加班管理 | 每月加班不超过36小时、足额支付加班费 | 未记录加班、以调休代替加班费 |

### 六、诉讼、仲裁与行政处罚

#### 审查 Checklist

- [ ] 通过裁判文书网、企查查等公开渠道检索目标公司涉诉信息
- [ ] 审查未决诉讼/仲裁的性质、金额及可能结果
- [ ] 核查行政处罚记录（环保、安全生产、税务、市场监管等）
- [ ] 评估或有负债（Contingent Liabilities）的规模
- [ ] 检查是否存在群体性诉讼风险
- [ ] 审查公司及实控人的征信报告
- [ ] 核查实控人是否被列入失信被执行人名单

### 七、行业合规与监管风险

#### 审查 Checklist

- [ ] 核查行业特定资质和许可证（如ICP、医疗器械经营许可证等）
- [ ] 审查数据合规性（《个人信息保护法》《数据安全法》《网络安全法》）
- [ ] 检查反垄断合规（经营者集中申报、垄断协议）
- [ ] 核查环境保护合规（排污许可、环评报告）
- [ ] 审查出口管制和制裁合规（如涉及国际贸易）
- [ ] 检查反商业贿赂和反腐败合规
- [ ] 审查广告宣传合规性

#### 重点行业合规要求

| 行业 | 核心监管要求 | 关键许可证 |
|------|------------|-----------|
| 金融科技 | 持牌经营、信息隔离、消费者保护 | 支付牌照、征信牌照、小贷牌照 |
| 医疗健康 | 临床试验、药品注册、GMP合规 | 药品经营许可证、医疗器械注册证 |
| 教育 | 营利性/非营利性分类管理 | 办学许可证（营利性需营业执照） |
| 数据服务 | 数据分类分级、跨境传输合规 | 数据安全评估通过 |
| 新能源 | 环保标准、安全生产 | 排污许可证、安全生产许可证 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 尽职调查 | Due Diligence (DD) | 投资前对目标公司进行全面法律调查和评估的过程 |
| 控制权变更条款 | Change of Control (CoC) | 合同中规定当公司控制权发生变化时对方有权终止或重新谈判的条款 |
| 代持 | Nominee Shareholding | 实际出资人委托他人以自己名义持有股权的安排 |
| VIE结构 | Variable Interest Entity | 通过协议控制而非股权控制实现对境内运营实体控制的架构 |
| 或有负债 | Contingent Liabilities | 取决于未来不确定事项是否发生的潜在债务 |
| 竞业限制 | Non-compete / Non-solicitation | 限制员工在职期间或离职后从事竞争性业务的约定 |
| 优先清算权 | Liquidation Preference | 投资人在公司清算或被收购时优先于普通股获得分配的权利 |
| 反稀释保护 | Anti-dilution Protection | 当后续融资价格低于前轮时，保护前轮投资人股权比例不被过度稀释的条款 |
| 领售权 | Drag-along Right | 允许特定股东在出售股权时强制其他股东一起出售的权利 |
| 随售权 | Tag-along Right | 当大股东出售股权时，小股东有权按同等条件参与出售的权利 |

## 实务要点

1. **穿透核查原则**：法律尽调不能停留在表面文件层面，需要穿透核查实际控制人、最终受益人，以及股权背后的真实利益关系。特别关注代持安排和隐性关联关系。

2. **关注合规经营的历史污点**：即使目标公司目前已规范经营，历史上的违规行为（如行政处罚、环保事故）也可能产生后续影响。需要评估这些历史问题的潜在法律后果。

3. **知识产权是核心资产**：对于科技类目标公司，知识产权往往是核心资产。需要特别关注IP的权属清晰性、保护范围和防御能力，以及团队成员的IP贡献和归属。

4. **劳动用工是隐形炸弹**：社保公积金欠缴、竞业限制瑕疵、劳动争议等看似小问题，在员工数量较多时可能累积成重大财务负担。建议按人头估算潜在补缴成本。

5. **合同中的"控制权变更"条款**：这是最容易被忽视但影响极大的条款。如果主要客户或供应商合同中包含CoC条款，投资交割后可能面临合同终止的风险，直接影响业务连续性。

## 常见误区

1. **只关注实体法律风险，忽视程序性风险**：很多投资人关注目标公司是否有诉讼，却忽视了公司治理程序的合规性（如股东会决议程序瑕疵、信息披露义务）。程序性瑕疵可能在后续融资或上市时成为实质性障碍。

2. **低估数据合规风险**：随着《个人信息保护法》《数据安全法》的实施，数据合规已成为科技企业的重大法律风险。违规处理个人信息可能面临高额罚款（最高年营业额5%），甚至刑事责任。

3. **忽视跨境法律风险**：涉及跨境业务的公司面临多法域合规要求（出口管制、制裁合规、数据跨境传输）。一个法域的合规问题可能牵连整个业务体系。
