# 税务筹划要点

## 概述
税务筹划是私募基金架构设计和运营管理中的核心议题。中国私募基金（特别是有限合伙型基金）的税务处理具有"穿透税制"的特点，即基金层面不征收所得税，由各合伙人/投资者分别纳税。合理利用税收政策和架构安排，可在合法合规的前提下显著降低投资人的税负。

## 核心内容

### 一、合伙企业穿透税制

#### 法律依据
- 《合伙企业法》（2006年修订）
- 《关于合伙企业合伙人所得税问题的通知》（财税〔2008〕159号）
- 《财政部、税务总局关于权益性投资经营所得个人所得税征收管理的公告》（财政部 税务总局公告2021年第41号）

#### 核心原则："先分后税"

合伙企业本身不是所得税纳税主体，其所得按照"先分后税"原则，由各合伙人分别缴纳所得税。

```
基金层面 → 不征收企业所得税
    ↓
穿透至合伙人
    ↓
自然人合伙人 → 缴纳个人所得税
法人合伙人 → 缴纳企业所得税
```

#### 各类合伙人税负对照

| 合伙人类型 | 所得类型 | 税率 | 适用政策 |
|------------|----------|------|----------|
| 自然人LP（股息红利） | 利息、股息、红利所得 | 20% | 个人所得税法 |
| 自然人LP（股权转让所得） | 经营所得 | 5%-35%累进 | 财税〔2019〕8号（单一基金核算可按20%） |
| 自然人GP（管理费收入） | 经营所得（劳务报酬性质） | 5%-35%累进 | 个人所得税法 |
| 自然人GP（超额收益/Carry） | 经营所得 | 5%-35%累进 | 财税〔2019〕8号（单一基金核算可按20%） |
| 法人LP | 企业所得 | 25% | 企业所得税法（合并至法人LP纳税） |
| 法人GP | 企业所得 | 25% | 企业所得税法 |

#### 创投基金税收优惠：70%投资额抵扣

**政策依据**：《关于创业投资企业和天使投资个人有关税收政策的通知》（财税〔2018〕55号）

**核心优惠：**
- 创业投资企业采取股权投资方式投资于未上市的中小高新技术企业2年以上的，可按投资额的70%在股权持有满2年的当年抵扣该创投企业的应纳税所得额
- 当年不足抵扣的，可在以后纳税年度结转抵扣

**适用条件：**
| 条件 | 要求 |
|------|------|
| 基金类型 | 完成备案的创业投资基金 |
| 投资对象 | 未上市的中小高新技术企业/初创科技型企业 |
| 持有期限 | 股权持有满2年（24个月） |
| 投资方式 | 股权投资（非债权） |
| 中小企业标准 | 年销售额/资产总额不超过2亿元，从业人员不超过500人 |
| 初创科技型企业 | 接受投资当年及下一纳税年度，研发费用占比不低于20% |

**实务计算示例：**
- 某创投基金投资1000万元于初创科技型企业，持有满2年
- 该基金当年股权转让所得为800万元
- 可抵扣金额 = 1000万 x 70% = 700万元
- 实际应纳税所得额 = 800万 - 700万 = 100万元
- 节税效果显著

### 二、GP/LP所得税处理详解

#### GP的税务处理

| 收入类型 | 税务处理 | 说明 |
|----------|----------|------|
| 管理费（通常2%） | 按"经营所得"缴纳个税（自然人）或企业所得税（法人） | 自然人GP适用5%-35%累进税率 |
| 超额收益（Carry，通常20%） | 同上 | 财税〔2019〕8号：选择单一基金核算的，按20%税率 |
| 跟投收益 | 按"财产转让所得"适用20%税率 | 视具体安排可能适用不同税率 |

#### LP的税务处理

| LP类型 | 股息红利所得 | 股权转让所得 |
|--------|-------------|-------------|
| 自然人LP | 20%（个人所得税） | 经营所得5%-35%或单一基金核算20% |
| 法人LP | 免税（居民企业间股息免税） | 25%企业所得税（合并计税） |
| 合伙企业LP | 穿透至最终合伙人 | 穿透至最终合伙人 |
| 有限合伙型创投LP | 20%（单一基金核算） | 20%（单一基金核算） |

#### 单一基金核算 vs 整体核算

**政策依据**：《关于创业投资企业个人合伙人所得税政策问题的通知》（财税〔2019〕8号）

| 核算方式 | 适用对象 | 税率 | 特点 |
|----------|----------|------|------|
| 单一基金核算 | 创投基金的个人合伙人 | 股权转让所得按20% | 不能用投资亏损抵扣其他所得 |
| 整体核算 | 一般基金的个人合伙人 | 经营所得5%-35% | 可以用投资亏损抵扣其他经营所得 |

### 三、转让定价风险

#### 关联交易定价的税务风险
私募基金运营中常见的关联交易场景：

| 场景 | 风险点 | 应对措施 |
|------|--------|----------|
| 管理费定价 | 管理费标准偏离市场公允水平 | 参照行业惯例（2%左右） |
| 关联方共同投资 | 被投企业利益向关联方输送 | 建立公允的投资决策流程 |
| 跨境利润分配 | 利润在低税率地区堆积 | 确保符合独立交易原则 |
| 技术服务费 | 以服务费名义转移利润 | 保留真实服务合同和发票 |

#### 税务机关关注的异常信号
- 长期亏损但持续经营
- 关联交易金额大但利润率异常低
- 向避税港支付大额费用
- 同期资料准备不充分

### 四、跨境投资税务架构

#### 典型架构设计

```
境内基金（有限合伙）
    ↓
香港中间公司（HK Co.）
    ↓ 投资
目标运营公司（境内/境外）
```

**香港中间公司的税务优势：**
- 香港利得税税率：8.25%（首200万港元）/ 16.5%
- 内地与香港税收安排（DTA）：股息预提税可降至5%（持股25%以上）或10%
- 无资本利得税（股权转让收益通常不征税）
- 香港不对境外来源所得征税（离岸所得免税申请）

#### 跨境投资主要税种

| 税种 | 纳税人 | 税率 | 适用场景 |
|------|--------|------|----------|
| 企业所得税 | 境内被投企业 | 25%（高新技术企业15%） | 企业经营利润 |
| 预提所得税 | 分配给境外投资者的股息 | 10%（协定优惠可降低） | 利润汇出 |
| 增值税 | 境内经营企业 | 6%/9%/13% | 商品和服务流转 |
| 印花税 | 股权转让方 | 0.05%（股权转让协议） | 股权交易 |

### 五、常见税务风险与应对

#### 风险一：自然人合伙人税负过重
- 问题：股权转让所得按经营所得适用5%-35%累进税率
- 应对：创投基金选择单一基金核算，享受20%优惠税率；考虑通过法人架构间接持有

#### 风险二：合伙企业亏损无法抵扣
- 问题：合伙企业亏损不能抵扣合伙人的其他所得
- 应对：合理安排投资节奏，避免大量投资集中在同一纳税年度

#### 风险三：税收优惠适用不当
- 问题：创投基金70%投资额抵扣的适用条件严格
- 应对：投资前确认被投企业是否符合中小高新技术企业/初创科技型企业标准，保留完整的证明材料

#### 风险四：跨境架构的反避税风险
- 问题：税务机关可能依据一般反避税条款（GAAR）否定不具有合理商业目的的跨境安排
- 应对：确保跨境架构具有合理的商业实质，而非仅为税务目的设立

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 穿透税制 | Pass-through Taxation | 合伙企业不作为纳税主体，所得穿透至各合伙人分别纳税 |
| 先分后税 | Distribute then Tax | 合伙企业的所得先按合伙协议分配，再由各合伙人分别纳税 |
| Carry | Carried Interest | 超额收益分成，通常为基金利润的20% |
| 单一基金核算 | Single Fund Accounting | 创投基金个人合伙人可选择按单一基金核算，享受20%税率 |
| 转让定价 | Transfer Pricing | 关联方之间交易的价格需符合独立交易原则 |
| GAAR | General Anti-Avoidance Rule | 一般反避税规则，赋予税务机关否定不合理避税安排的权力 |
| DTA | Double Taxation Agreement | 双边税收协定，避免双重征税 |
| 预提所得税 | Withholding Tax | 对支付给非居民的所得源泉扣缴的所得税 |

## 实务要点
- 基金架构设计阶段就应进行税务测算，比较不同架构（有限合伙 vs 公司制 vs 契约制）下的综合税负
- 创投基金应尽早选择"单一基金核算"方式，以享受20%的优惠税率，但需注意选择后36个月内不得变更
- 跨境投资架构中，应充分利用税收协定优惠（如中港税收安排），但须满足"受益所有人"（Beneficial Owner）条件
- 建立完善的关联交易文档和同期资料，防范转让定价调整风险
- 关注各地方税收优惠政策（如海南自由贸易港、横琴粤澳深度合作区等），但需注意政策的稳定性和执行口径

## 常见误区
- **误区一**："合伙企业不用交税"——合伙企业虽然不是所得税纳税主体，但仍需进行纳税申报，且各合伙人需分别缴纳所得税
- **误区二**："所有私募基金都可以享受创投基金70%抵扣优惠"——只有符合特定条件的创业投资基金（投资于未上市中小高新技术企业/初创科技型企业）才能享受
- **误区三**："设在低税率地区的基金一定税负低"——基金注册地不改变穿透税制下合伙人的纳税义务，自然人合伙人的经营所得仍适用5%-35%累进税率
