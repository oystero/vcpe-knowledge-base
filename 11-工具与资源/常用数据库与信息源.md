# 常用数据库与信息源

## 概述
VC/PE投资研究依赖高质量的数据和信息支持，从项目发现、尽职调查到投后管理、业绩分析，每个环节都需要可靠的数据来源。了解并熟练使用各类专业数据库和信息平台，是提升投资研究效率和决策质量的基础。本文系统梳理VC/PE从业者常用的数据库、信息源和研究工具。

## 核心内容

### 一、一级市场数据平台

#### 1. 中国市场数据平台

**IT桔子（itjuzi.com）**

| 维度 | 说明 |
|------|------|
| 数据覆盖 | 中国创业公司、投资事件、投资机构 |
| 核心功能 | 融资事件追踪、公司画像、行业分析 |
| 更新频率 | 实时更新 |
| 价格区间 | 基础版免费，专业版约5,000-20,000元/年 |
| 适用场景 | 中国市场投资研究、行业趋势分析 |

**优势**：中国创业公司数据最全面，融资事件覆盖广
**局限**：部分数据准确性有待验证，需要交叉核实

**天眼查（tianyancha.com）**

| 维度 | 说明 |
|------|------|
| 数据覆盖 | 中国企业工商信息、司法信息、知识产权 |
| 核心功能 | 企业背景调查、股权穿透、关联关系图谱 |
| 更新频率 | 实时更新 |
| 价格区间 | 基础版免费，VIP约3,000-10,000元/年 |
| 适用场景 | 尽职调查、风险排查、竞品分析 |

**优势**：工商数据权威，股权穿透功能强大
**局限**：主要覆盖中国市场，海外数据有限

**企查查（qcc.com）**

| 维度 | 说明 |
|------|------|
| 数据覆盖 | 与天眼查类似，覆盖中国企业信息 |
| 核心功能 | 企业查询、风险监控、产业链分析 |
| 更新频率 | 实时更新 |
| 价格区间 | 基础版免费，VIP约3,000-10,000元/年 |
| 适用场景 | 尽职调查、企业信用评估 |

**优势**：API接口友好，适合批量查询
**局限**：部分增值服务收费较高

#### 2. 全球市场数据平台

**PitchBook（pitchbook.com）**

| 维度 | 说明 |
|------|------|
| 数据覆盖 | 全球PE、VC、M&A交易数据 |
| 核心功能 | 投资事件、基金业绩、估值数据、行业分析 |
| 更新频率 | 实时更新 |
| 价格区间 | 约$15,000-$40,000/年（企业版） |
| 适用场景 | 全球PE/VC市场研究、业绩基准分析 |

**优势**：全球PE/VC数据最权威，基金业绩数据详尽
**局限**：价格昂贵，中小企业/个人用户难以承受

**Preqin（preqin.com）**

| 维度 | 说明 |
|------|------|
| 数据覆盖 | 全球另类资产数据（PE、VC、房地产、基础设施等） |
| 核心功能 | 基金数据库、LP数据库、业绩基准、现金流分析 |
| 更新频率 | 实时更新 |
| 价格区间 | 约$10,000-$30,000/年 |
| 适用场景 | LP投资决策、GP募资研究、业绩对标 |

**优势**：LP/GP关系数据全面，业绩基准权威
**局限**：部分发展市场数据覆盖不完整

**Crunchbase（crunchbase.com）**

| 维度 | 说明 |
|------|------|
| 数据覆盖 | 全球创业公司、投资事件、投资机构 |
| 核心功能 | 公司发现、融资追踪、行业趋势 |
| 更新频率 | 实时更新 |
| 价格区间 | 基础版免费，Pro版约$29-$99/月 |
| 适用场景 | 创业公司研究、投资机会发现 |

**优势**：免费版功能实用，全球创业公司数据丰富
**局限**：深度数据需要付费，业绩数据有限

## 实务要点

- **建立多源交叉验证机制**：重要数据至少从两个独立来源核实
- **关注数据更新频率**：选择更新及时的数据平台，避免使用过时数据
- **平衡免费与付费资源**：充分利用免费资源，对核心需求投入付费服务
- **培养信息筛选能力**：建立信息过滤机制，避免信息过载
- **建立个人知识库**：系统整理和归档有价值的信息，便于检索复用

## 常见误区

- **过度依赖单一数据源**：任何单一数据平台都可能存在错误或偏差，需要交叉验证
- **忽视数据局限性**：免费数据往往覆盖不全、准确性不足，需要了解其局限
- **信息收集替代分析**：数据本身不产生价值，关键在于分析和应用
