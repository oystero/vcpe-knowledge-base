# 财务模型模板说明

## 概述
财务模型是VC/PE投资决策的核心工具，通过量化分析帮助投资人评估企业价值、测算投资回报、进行敏感性分析。不同投资场景需要不同类型的财务模型，掌握主要模型的搭建方法和应用要点，是投资从业者的基本技能。本文介绍VC/PE领域最常用的三种财务模型及其搭建要点。

## 核心内容

### 一、常用财务模型类型

#### 1. DCF模型（Discounted Cash Flow，现金流折现模型）

**核心原理**
- 通过预测企业未来自由现金流，以适当的折现率折算为现值
- 理论上最"正确"的估值方法，基于企业内在价值
- 对假设输入高度敏感，需要谨慎选择参数

**适用场景**
- 现金流可预测的成熟企业
- 基础设施、公用事业等稳定行业
- 长期战略投资评估

**DCF模型结构**

```
┌─────────────────────────────────────────────────────┐
│                   DCF模型框架                         │
├─────────────────────────────────────────────────────┤
│  历史财务数据（3-5年）                                │
│     ↓                                                │
│  收入预测（市场规模 × 市场份额 × 客单价）              │
│     ↓                                                │
│  成本费用预测（固定成本 + 变动成本）                   │
│     ↓                                                │
│  自由现金流（FCF = EBITDA - 资本支出 - 营运资金变动）  │
│     ↓                                                │
│  折现率计算（WACC）                                   │
│     ↓                                                │
│  终值计算（Terminal Value）                           │
│     ↓                                                │
│  企业价值（EV）= PV(FCF) + PV(TV)                    │
│     ↓                                                │
│  股权价值 = EV - 净债务                               │
└─────────────────────────────────────────────────────┘
```

**关键假设输入**

| 假设类别 | 具体参数 | 数据来源 |
|----------|----------|----------|
| 收入增长 | 年增长率、增长驱动因素 | 行业报告、管理层指引 |
| 利润率 | 毛利率、EBITDA利润率 | 历史数据、可比公司 |
| 资本支出 | 维护性/扩张性资本支出 | 历史数据、管理层指引 |
| 营运资金 | 周转天数、占收入比例 | 历史数据、行业基准 |
| 折现率 | WACC | CAPM模型计算 |
| 终值假设 | 永续增长率、退出倍数 | 长期GDP增长、行业经验 |

**WACC计算公式**

$$WACC = \frac{E}{E+D} \times R_e + \frac{D}{E+D} \times R_d \times (1-T)$$

其中：
- E = 股权市值
- D = 债务市值
- R_e = 股权成本（CAPM计算）
- R_d = 债务成本
- T = 税率

**DCF模型的局限性**

| 局限 | 说明 | 应对方法 |
|------|------|----------|
| 假设敏感性 | 小幅假设变化导致大幅估值波动 | 敏感性分析、情景分析 |
| 终值占比高 | 终值可能占估值50%以上 | 测试不同终值假设 |
| 预测不确定性 | 远期预测准确度低 | 使用蒙特卡洛模拟 |
| 不适用早期企业 | 早期企业现金流不可预测 | 改用相对估值法 |

#### 2. LBO模型（Leveraged Buyout Model，杠杆收购模型）

**核心原理**
- 通过债务杠杆放大股权投资回报
- 关注企业现金流对债务的覆盖能力
- PE并购交易的标准分析工具

**适用场景**
- PE并购交易估值
- 现金流稳定的成熟企业
- 有明确退出路径的投资

**LBO模型结构**

```
┌─────────────────────────────────────────────────────┐
│                   LBO模型框架                         │
├─────────────────────────────────────────────────────┤
│  交易结构假设                                          │
│  ├─ 收购价格（EV/EBITDA倍数）                         │
│  ├─ 资本结构（债务/股权比例）                          │
│  ├─ 债务条款（利率、期限、还款计划）                    │
│  └─ 交易费用                                          │
│     ↓                                                │
│  被投企业财务预测                                      │
│  ├─ 收入增长                                          │
│  ├─ 利润率变化                                        │
│  ├─ 资本支出                                          │
│  └─ 营运资金                                          │
│     ↓                                                │
│  现金流与债务分析                                      │
│  ├─ 自由现金流                                        │
│  ├─ 债务偿还计划                                      │
│  └─ 覆盖比率（DSCR、ICR）                            │
│     ↓                                                │
│  退出分析                                             │
│  ├─ 退出时间（通常3-5年）                             │
│  ├─ 退出倍数                                          │
│  └─ 退出价值                                          │
│     ↓                                                │
│  回报测算                                             │
│  ├─ 股权IRR                                           │
│  ├─ 股权倍数（MOIC）                                  │
│  └─ 敏感性分析                                        │
└─────────────────────────────────────────────────────┘
```

**LBO关键指标**

| 指标 | 计算公式 | 健康范围 |
|------|----------|----------|
| DSCR | EBITDA / 债务本息 | >1.5x |
| ICR | EBITDA / 利息费用 | >3.0x |
| 杠杆倍数 | 债务 / EBITDA | 3-6x |
| 股权占比 | 股权 / 总对价 | 30-50% |

**LBO模型的特殊假设**

| 假设类别 | 具体参数 | 经验值 |
|----------|----------|--------|
| 收购倍数 | EV/EBITDA | 8-15x（行业差异大） |
| 债务成本 | 优先债/次级债利率 | 5-12% |
| 退出倍数 | 通常等于或略低于收购倍数 | 8-12x |
| 持有期 | 3-5年 | PE标准周期 |

#### 3. 可比公司分析模型（Comparable Company Analysis）

**核心原理**
- 通过与类似公司的市场估值对比，推算目标公司价值
- 基于市场定价，反映当前市场情绪
- 相对简单快速，但依赖可比公司的选择

**适用场景**
- IPO定价参考
- 并购交易估值
- 快速估值验证
- 缺乏详细财务预测时

**可比公司分析框架**

```
┌─────────────────────────────────────────────────────┐
│              可比公司分析框架                          │
├─────────────────────────────────────────────────────┤
│  Step 1: 选择可比公司                                 │
│  ├─ 行业相同或相似                                    │
│  ├─ 规模相近                                          │
│  ├─ 增长阶段类似                                      │
│  └─ 地理市场类似                                      │
│     ↓                                                │
│  Step 2: 收集估值倍数                                 │
│  ├─ EV/Revenue                                        │
│  ├─ EV/EBITDA                                         │
│  ├─ P/E                                               │
│  └─ P/B                                               │
│     ↓                                                │
│  Step 3: 计算可比公司估值区间                          │
│  ├─ 中位数                                            │
│  ├─ 平均数                                            │
│  └─ 25%-75%分位数                                     │
│     ↓                                                │
│  Step 4: 应用于目标公司                                │
│  ├─ 选择适当的倍数区间                                 │
│  ├─ 考虑目标公司相对优劣势调整                         │
│  └─ 计算企业价值/股权价值                              │
└─────────────────────────────────────────────────────┘
```

**常用估值倍数**

| 倍数 | 计算公式 | 适用场景 |
|------|----------|----------|
| EV/Revenue | 企业价值 / 收入 | 高增长、亏损企业 |
| EV/EBITDA | 企业价值 / EBITDA | 盈利稳定企业 |
| P/E | 股价 / 每股收益 | 成熟上市公司 |
| P/B | 股价 / 每股净资产 | 资产密集型行业 |
| EV/Subscriber | 企业价值 / 用户数 | 互联网/平台企业 |

**可比公司选择标准**

| 标准 | 权重 | 说明 |
|------|------|------|
| 行业一致性 | 高 | 主营业务相同或高度相似 |
| 规模可比性 | 中 | 收入/市值在同一数量级 |
| 增长阶段 | 中 | 处于相似的生命周期阶段 |
| 盈利能力 | 中 | 利润率水平相近 |
| 地域市场 | 低 | 服务的地理市场类似 |

**Sheet 2：利润表（Income Statement）**

| 项目 | 预测逻辑 | 数据来源 |
|------|----------|----------|
| 营业收入 | 市场规模 × 市场份额 | 假设页 |
| 营业成本 | 收入 × (1-毛利率) | 假设页 |
| 毛利润 | 收入 - 成本 | 计算 |
| 运营费用 | 收入 × 费用率 | 假设页 |
| EBITDA | 毛利润 - 运营费用（不含折旧） | 计算 |
| 折旧摊销 | 固定资产/无形资产摊销 | 假设页 |
| EBIT | EBITDA - 折旧摊销 | 计算 |
| 利息费用 | 债务余额 × 利率 | 假设页 |
| 税前利润 | EBIT - 利息费用 | 计算 |
| 所得税 | 税前利润 × 税率 | 假设页 |
| 净利润 | 税前利润 - 所得税 | 计算 |

**Sheet 3：资产负债表（Balance Sheet）**

| 科目 | 预测逻辑 |
|------|----------|
| 流动资产 | 现金、应收账款、存货（基于周转天数） |

| 流动负债 | 应付账款、短期借款（基于周转天数） |
| 非流动负债 | 长期借款（基于还款计划） |
| 股东权益 | 留存收益累计 |

**Sheet 4：现金流量表（Cash Flow Statement）**

| 类别 | 主要项目 |

| 投资活动 | 资本支出、投资收益 |

### 三、关键假设输入

**收入预测假设**

| 假设层级 | 具体参数 | 验证方法 |
|----------|----------|----------|
| 宏观层 | GDP增长、行业增速 | 权威机构预测 |
| 市场层 | 市场规模、渗透率 | 行业报告 |
| 公司层 | 市场份额、客单价 | 管理层指引、历史数据 |

**成本预测假设**

| 成本类型 | 预测方法 | 注意事项 |
|----------|----------|----------|
| 固定成本 | 基于人员规模和租金 | 关注规模效应 |
| 变动成本 | 收入 × 变动成本率 | 关注边际成本变化 |
| 资本支出 | 维护性 + 扩张性 | 区分必要和可选支出 |

**估值参数假设**

| 参数 | 计算方法 | 数据来源 |
|------|----------|----------|
| 无风险利率 | 10年期国债收益率 | 市场数据 |
| 股权风险溢价 | 历史平均 + 调整 | 学术研究、咨询公司 |
| Beta | 可比公司回归分析 | Wind/Bloomberg |
| 债务成本 | 市场利率 + 信用利差 | 信用评级、市场数据 |

### 四、敏感性分析

**敏感性分析的目的**
- 识别对估值/回报影响最大的假设
- 测试不同情景下的投资可行性
- 支持投资决策的风险评估

**单变量敏感性分析**

改变单一假设，观察对结果的影响：

| 假设变量 | 基准值 | -20% | -10% | 基准 | +10% | +20% |
|----------|--------|------|------|------|------|------|
| 收入增长率 | 25% | X1 | X2 | X3 | X4 | X5 |
| EBITDA利润率 | 20% | Y1 | Y2 | Y3 | Y4 | Y5 |
| 退出倍数 | 10x | Z1 | Z2 | Z3 | Z4 | Z5 |

**双变量敏感性分析**

同时改变两个假设，观察交叉影响：

| | 退出倍数 8x | 退出倍数 10x | 退出倍数 12x |
|---|-------------|--------------|--------------|
| 收入增长20% | A1 | A2 | A3 |
| 收入增长25% | B1 | B2 | B3 |
| 收入增长30% | C1 | C2 | C3 |

**情景分析**

| 情景 | 收入增长 | 利润率 | 退出倍数 | IRR |
|------|----------|--------|----------|-----|
| 乐观 | 35% | 25% | 12x | 40% |
| 基准 | 25% | 20% | 10x | 28% |
| 悲观 | 15% | 15% | 8x | 15% |

### 五、模型搭建要点

**模型设计原则**

| 原则 | 说明 | 实践建议 |
|------|------|----------|
| 清晰性 | 结构清晰，易于理解 | 使用统一的格式和颜色 |
| 灵活性 | 假设易于调整 | 所有假设集中在假设页 |
| 一致性 | 公式逻辑一致 | 避免硬编码，使用公式引用 |
| 可追溯性 | 公式来源清晰 | 使用注释和说明 |
| 稳健性 | 错误检查机制 | 添加错误检查公式 |

**常见错误与防范**

| 错误类型 | 具体表现 | 防范措施 |
|----------|----------|----------|
| 循环引用 | 现金流和利息互为因果 | 使用迭代计算或结构调整 |
| 硬编码 | 直接输入数字而非引用 | 所有数字链接到假设页 |
| 单位错误 | 千元/百万元混用 | 统一单位，添加单位标注 |
| 时间错误 | 年度/季度数据混用 | 清晰标注时间维度 |
| 基准年错误 | 预测起点错误 | 核实基准年数据 |

**模型检查清单**

```
□ 假设页是否包含所有关键假设
□ 历史数据是否准确可靠
□ 预测逻辑是否合理
□ 利润表、资产负债表、现金流量表是否勾稽平衡
□ 估值方法是否适用
□ 敏感性分析是否覆盖关键变量
□ 单位是否统一
□ 公式是否可追溯
□ 是否有错误检查机制
□ 图表是否清晰展示关键结论
```

### 六、VC早期项目估值的特殊性

**传统估值方法的局限**

早期VC项目通常：
- 没有历史财务数据
- 现金流高度不确定
- 难以找到可比公司
- 传统DCF不适用

**VC常用估值方法**

| 方法 | 原理 | 适用阶段 |
|------|------|----------|
| Berkus方法 | 基于5个关键因素打分估值 | 种子轮 |
| Scorecard方法 | 与可比公司对比调整 | 天使轮 |
| 风险因素法 | 识别并量化各类风险 | 天使/Pre-A |
| 可比交易法 | 参考类似交易估值 | A轮及以后 |
| 收入倍数法 | 收入 × 行业倍数 | 有收入后 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| DCF | Discounted Cash Flow | 现金流折现估值方法 |
| LBO | Leveraged Buyout | 杠杆收购 |
| WACC | Weighted Average Cost of Capital | 加权平均资本成本 |
| EBITDA | Earnings Before Interest, Taxes, Depreciation, and Amortization | 息税折旧摊销前利润 |
| FCFF | Free Cash Flow to Firm | 企业自由现金流 |
| Terminal Value | 终值 | 预测期末的企业价值 |
| Sensitivity Analysis | 敏感性分析 | 测试假设变化对结果的影响 |
| MOIC | Multiple on Invested Capital | 投资回报倍数 |

## 实务要点

- **假设要有依据**：每个关键假设都应有数据支持或合理逻辑
- **模型要灵活**：假设应易于调整，便于快速测试不同情景
- **关注关键驱动因素**：识别对结果影响最大的假设，重点验证
- **多方法交叉验证**：使用2-3种估值方法，取合理区间
- **定期更新**：根据最新信息及时更新模型假设

## 常见误区

- **过度精确**：对不确定的预测追求过高精度，忽视假设本身的不确定性
- **忽视终值**：终值可能占估值50%以上，但往往被草率处理
- **模型代替判断**：模型只是决策支持工具，不能替代投资判断
