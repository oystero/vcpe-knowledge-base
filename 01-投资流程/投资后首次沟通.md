# 投资后首次沟通

## 概述
投资完成后首次深度沟通是投后管理的起点，决定了投资人与被投企业之间的合作基调。首次董事会/股东会、90 天行动计划和 KPI 基线建立是这一阶段的三大核心任务。好的开始能建立信任、明确期望、为后续长期合作奠定基础。

## 核心内容

### 首次沟通的时间窗口

| 时间节点 | 核心任务 | 参与方 |
|----------|----------|--------|
| 交割后 1 周 | 非正式沟通（电话/面谈） | 投资人 + 创始人 |
| 交割后 2-3 周 | 首次董事会/股东会 | 全体股东、董事 |
| 交割后 30 天 | 90 天行动计划确认 | 投资人 + 管理层 |
| 交割后 90 天 | 首次季度回顾 | 投资人 + 管理层 |

### 交割后首次非正式沟通

**目的：**
- 建立个人层面的信任关系
- 了解创始人的真实期望和担忧
- 明确投资人的角色定位（增值而非干预）

**核心议题：**
1. **创始人近况：** 团队士气、创始人的个人状态和压力点
2. **公司现状：** 交割期间是否有新的变化（客户、竞争、团队）
3. **资源需求：** 创始人最希望投资人提供哪些帮助
4. **沟通偏好：** 确定日常沟通的频率、方式和对接人
5. **近期计划：** 接下来 1-3 个月的核心工作重点

### 首次董事会 / 股东会

#### 会议准备

| 准备事项 | 负责方 | 时间要求 |
|----------|--------|----------|
| 会议通知 | 公司秘书 | 提前 15 天 |
| 议程确认 | 投资人 + 创始人 | 提前 10 天 |
| 财务报告 | 公司 CFO | 提前 7 天 |
| 战略报告 | 创始人/CEO | 提前 7 天 |
| 会议材料分发 | 公司秘书 | 提前 5 天 |

#### 首次董事会议程模板

| 议题 | 时间（分钟） | 说明 |
|------|-------------|------|
| 开场与介绍 | 10 | 新任董事介绍、角色确认 |
| 公司概况更新 | 20 | CEO 介绍公司现状、业务进展 |
| 财务回顾 | 30 | CFO 介绍最新财务数据和现金流 |
| 战略讨论 | 40 | 未来 12 个月的战略规划和优先事项 |
| 融资使用计划 | 15 | 本轮资金的使用计划和里程碑 |
| 治理安排 | 20 | 确认董事会运作规则、报告机制 |
| KPI 基线讨论 | 20 | 确认核心监控指标和报告频率 |
| 其他事项 | 15 | 其他需要讨论的事项 |
| **合计** | **约 3 小时** | |

#### 首次董事会的关键产出

- **董事会决议：** 选举董事、批准公司章程、确认治理安排
- **KPI 框架：** 确认核心监控指标和报告频率
- **90 天行动计划：** 确认近三个月的优先事项和里程碑
- **沟通机制：** 确定定期沟通的频率和方式

### 90 天行动计划

90 天计划是投资后第一个季度的行动框架，帮助双方在短期内建立合作节奏。

#### 90 天计划模板

| 时间 | 优先事项 | 里程碑 | 责任人 |
|------|----------|--------|--------|
| **第 1 个月** | | | |
| 团队稳定 | 核心人员留任确认 | 100% 核心人员签署确认函 | CEO |
| 资金使用 | 按计划使用融资资金 | 首笔资金投入核心项目 | CFO |
| 战略对齐 | 与投资人对齐战略方向 | 战略共识文档完成 | CEO + 投资人 |
| **第 2 个月** | | | |
| 业务推进 | 核心业务指标改善 | 关键 KPI 达到阶段性目标 | 业务负责人 |
| 人才补充 | 关键岗位招聘 | 至少 1 个关键岗位到岗 | HR |
| 资源对接 | 投资人资源引入 | 至少 2 个有价值的商业对接 | 投资人 |
| **第 3 个月** | | | |
| 季度回顾 | 首次季度业绩回顾 | 季度报告完成 | CFO |
| 下季度规划 | 下季度 OKR/KPI 设定 | OKR 文档完成 | CEO |
| 反馈机制 | 建立双向反馈机制 | 首次满意度评估 | 双方 |

### KPI 基线建立

KPI 基线是投资时点的核心业务指标快照，作为后续业绩评估的基准。

#### 通用 KPI 框架

| KPI 类别 | 核心指标 | 报告频率 |
|----------|----------|----------|
| **财务指标** | | |
| 收入 | MRR/ARR、收入增长率、收入构成 | 月度 |
| 盈利 | 毛利率、EBITDA、净利率 | 月度 |
| 现金 | 现金余额、Burn Rate、Runway | 月度 |
| **运营指标** | | |
| 客户 | 客户数、客户获取成本（CAC）、客户生命周期价值（LTV） | 月度 |
| 产品 | DAU/MAU、留存率、NPS | 月度 |
| 效率 | 人效比（收入/员工数）、单位经济模型 | 季度 |
| **增长指标** | | |
| 销售 | Pipeline、成交率、平均客单价 | 月度 |
| 市场 | 市场份额、品牌知名度 | 季度 |
| 产品 | 新功能上线率、技术债务 | 月度 |
| **团队指标** | | |
| 人员 | 员工数、关键岗位到岗率、离职率 | 月度 |
| 文化 | 员工满意度、敬业度 | 季度 |

#### 不同阶段的 KPI 重点

| 公司阶段 | 核心 KPI | 次要 KPI |
|----------|----------|----------|
| 种子期 | 产品验证（用户反馈、MVP 迭代速度） | 团队建设、资金使用效率 |
| A 轮 | 用户增长、留存率、单位经济模型 | 收入（如有）、融资进度 |
| B 轮 | 收入增长、毛利率、CAC/LTV | 市场份额、运营效率 |
| C 轮+ | 收入规模、盈利能力、现金流 | 行业地位、国际化进展 |
| PE 阶段 | EBITDA、现金流、ROIC | 行业整合、运营优化 |

### 投后沟通机制

#### 定期沟通安排

| 沟通形式 | 频率 | 内容 | 参与方 |
|----------|------|------|--------|
| 月度报告 | 每月 | 财务数据 + 业务指标 + 关键事项 | 投资人 + CFO |
| 季度董事会 | 每季度 | 深度业绩回顾 + 战略讨论 | 全体董事 |
| 年度股东会 | 每年 | 年度业绩 + 审计报告 + 重大决议 | 全体股东 |
| 日常沟通 | 随时 | 紧急事项、资源对接、建议咨询 | 投资人 + CEO |
| 投后走访 | 每季度 | 现场考察、团队交流 | 投资人 |

#### 信息报告模板（月度）

| 板块 | 内容 |
|------|------|
| 关键指标概览 | 本月 vs 上月 vs 预算的核心 KPI 对比 |
| 重大事项 | 本月发生的重大事件（客户、产品、团队、竞争） |
| 资金使用 | 本月资金使用情况、剩余 Runway |
| 下月计划 | 下个月的核心工作重点 |
| 需要协助事项 | 需要投资人提供的资源或支持 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 投后管理 | Portfolio Management / Post-Investment Management | 投资完成后对被投企业的管理和增值服务 |
| 月度经常性收入 | MRR (Monthly Recurring Revenue) | SaaS 等订阅制业务的月度经常性收入 |
| 年度经常性收入 | ARR (Annual Recurring Revenue) | MRR x 12，SaaS 业务的核心收入指标 |
| 客户获取成本 | CAC (Customer Acquisition Cost) | 获取一个新客户的平均成本 |
| 客户生命周期价值 | LTV (Lifetime Value) | 一个客户在整个生命周期内贡献的收入/利润 |
| 烧钱率 | Burn Rate | 公司每月消耗现金的速度 |
| 现金跑道 | Runway | 按当前 Burn Rate 计算的现金可支撑月数 |
| 净推荐值 | NPS (Net Promoter Score) | 衡量客户满意度和推荐意愿的指标 |

## 实务要点

- **首次沟通要"以倾听为主"：** 投资人在首次深度沟通中应更多地了解创始人的想法和需求，而非急于"指导"或"施加影响"
- **KPI 基线要在投资时就建立：** 不要等到投后才开始收集数据，投资 DD 阶段就应记录核心指标作为基线
- **沟通频率要匹配公司阶段：** 早期项目需要更频繁的沟通（每周/每两周），成熟项目可以按月/季度
- **增值服务要有边界感：** 投资人的角色是"帮忙不添乱"，资源对接和战略建议为主，避免越界干预日常运营
- **建立双向反馈机制：** 不仅投资人要评价公司表现，创始人也应有机会反馈投资人的增值服务效果

## 常见误区

- **投后管理就是"要报表"：** 如果投后管理仅限于定期收集财务数据，就失去了增值的意义，投资人应主动参与战略讨论和资源对接
- **过度干预日常运营：** 投资人应尊重创始人的运营自主权，特别是在投资人不熟悉的领域，过度干预会破坏信任关系
- **忽视创始人的情绪和压力：** 创业过程中创始人面临巨大压力，投资人除了关注业务指标，也应关注创始人的状态和需求
