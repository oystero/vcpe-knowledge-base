# 项目来源与筛选

## 概述
Deal Flow（项目来源）是 VC/PE 投资的起点，高质量的 Deal Flow 是基金超额回报的基础。项目筛选通过系统化的漏斗模型将大量机会转化为少量高质量投资，筛选效率直接决定了基金的投资产出比。

## 核心内容

### Deal Flow 来源渠道

| 来源渠道 | 占比（典型） | 项目质量 | 特点 |
|----------|-------------|---------|------|
| FA（财务顾问） | 25%-35% | 中高 | 项目经过初步包装，竞争激烈 |
| 行业研究驱动 | 15%-25% | 高 | 主动发现，竞争较少，但耗时 |
| 老股东/被投企业推荐 | 15%-25% | 高 | 信息优势明显，已知企业运营情况 |
| 孵化器/加速器 | 5%-15% | 中 | 早期项目为主，数量多质量参差 |
| 创业者主动联系 | 5%-10% | 低-中 | 需要大量筛选，偶有惊喜 |
| 同行/GP 互荐 | 5%-10% | 中高 | 项目已在其他基金视野中 |
| 产业方/CVC 推荐 | 5%-10% | 中高 | 带有产业视角，但可能有战略绑定 |

### 各渠道详解

#### 1. FA（Financial Advisor / 财务顾问）

**国内主要 FA 机构（2024-2025）：**
- 华兴资本（China Renaissance）—— 互联网/TMT 领域头部
- 泰合资本（Alliance Capital Partners）—— 成长期项目
- 光源资本（Lighthouse Capital）—— 新经济领域
- 浩悦资本（Hao Yue Capital）—— 医疗/消费领域
- 指南投资（CS Capital）—— 硬科技/半导体

**与 FA 合作的要点：**
- FA 会同时推送给多家基金，项目竞争窗口通常 2-4 周
- 关注 FA 的专业度和行业覆盖，好的 FA 能提供深度行业 insight
- FA 费用通常由被投企业承担（融资金额的 1%-5%）

#### 2. 行业研究驱动（Research-Driven Sourcing）

**系统化方法：**
- 行业 Map 绘制：按细分赛道梳理产业链，标注关键企业
- 数据驱动筛选：利用 Crunchbase、IT 桔子、天眼查等数据库筛选
- 行业会议/展会：参加垂直行业峰会（如世界人工智能大会、BIO 生物医药大会）
- 学术/技术跟踪：关注顶级期刊和专利数据库，发现技术拐点

#### 3. 老股东/被投企业推荐

**为什么这是最优质的来源：**
- 已了解企业运营情况，信息不对称最低
- 被投企业创始人通常有行业人脉，推荐质量高
- 可能获得优先投资权或更优惠的条款
- 有助于构建 Portfolio Network 效应

### 项目筛选漏斗模型

```
                    ┌─────────────────────────────┐
                    │   初步接触（Initial Contact）  │  1,000 个项目/年
                    └──────────────┬──────────────┘
                                   ▼
                    ┌─────────────────────────────┐
                    │   初步筛选（Initial Screen）   │  200-300 个
                    │   行业匹配 + 基本面快筛        │
                    └──────────────┬──────────────┘
                                   ▼
                    ┌─────────────────────────────┐
                    │   深度讨论（Deep Dive）        │  50-80 个
                    │   团队访谈 + 行业研究          │
                    └──────────────┬──────────────┘
                                   ▼
                    ┌─────────────────────────────┐
                    │   尽职调查（DD）               │  15-25 个
                    │   财务/法律/商业/技术 DD        │
                    └──────────────┬──────────────┘
                                   ▼
                    ┌─────────────────────────────┐
                    │   投决会（IC）                 │  8-15 个
                    │   投资决策 + 条款谈判          │
                    └──────────────┬──────────────┘
                                   ▼
                    ┌─────────────────────────────┐
                    │   交割完成（Closing）          │  5-10 个
                    └─────────────────────────────┘
```

**关键比率指标：**
- 接触到初筛通过率：20%-30%
- 初筛到深度讨论率：20%-25%
- 深度讨论到 DD 率：30%-40%
- DD 到投决通过率：50%-60%
- 投决到交割完成率：70%-80%
- **总体转化率：约 0.5%-1%（每接触 100 个项目投资 1 个）**

### 初步筛选标准（Screening Criteria）

| 评估维度 | 关键问题 | 权重 |
|----------|----------|------|
| 市场空间 | TAM 是否足够大（>100 亿）？增长率如何？ | 20% |
| 商业模式 | 是否有清晰的盈利模式？单位经济模型是否成立？ | 15% |
| 团队 | 创始人背景、行业经验、团队完整性 | 25% |
| 产品/技术 | 产品差异化、技术壁垒、产品成熟度 | 15% |
| 竞争格局 | 市场集中度、竞争壁垒、护城河 | 10% |
| 阶段匹配 | 项目阶段是否匹配基金策略（种子/VC/PE） | 10% |
| 估值合理性 | 当前估值是否在可接受范围内 | 5% |

### 高效筛选工具

**一票否决条件（Red Flags）：**
- 创始人诚信问题（虚假数据、重大隐瞒）
- 市场空间过小（TAM < 10 亿人民币）
- 商业模式存在根本性缺陷（无盈利路径）
- 核心团队严重不完整且创始人无补全意愿
- 估值严重偏离基本面（>2x 合理估值）
- 与基金投资策略严重不匹配

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 项目来源 | Deal Flow | 投资机构接触到的投资机会总量 |
| 财务顾问 | FA (Financial Advisor) | 帮助企业融资的专业中介机构 |
| 漏斗模型 | Funnel / Pipeline | 从大量机会中筛选出少量投资的系统化流程 |
| 初步筛选 | Initial Screen / First Screen | 对项目进行快速评估和筛选的过程 |
| 深度讨论 | Deep Dive / Deep Due Diligence | 对项目进行深入研究和评估的阶段 |
| 行业地图 | Industry Map / Landscape | 对某一行业细分领域的全景式梳理 |
| 一票否决 | Knock-Out Criteria | 任何一条不满足即终止评估的硬性标准 |

## 实务要点

- **构建多元化的 Deal Flow 渠道：** 不要过度依赖单一来源，理想的比例是主动发现（行业研究 + 老股东推荐）占 40% 以上，被动接收（FA + 创业者联系）占 60% 以下
- **建立系统化的 CRM 管理：** 使用 Deal Flow 管理工具（如 Affinity、DealCloud 或自建系统）追踪每个项目的来源、状态和评估记录
- **初筛要快但不要草率：** 初筛阶段应在 1-2 天内完成，但要确保覆盖核心维度（市场、团队、商业模式），避免因"怕错过"而浪费后续 DD 资源
- **重视"非共识"项目：** 最好的投资往往来自其他基金忽视的领域或团队，过于依赖 FA 热门项目会导致竞争加剧和估值泡沫
- **定期复盘筛选漏斗数据：** 每季度分析各渠道的转化率、项目质量和投资产出，持续优化 Sourcing 策略

## 常见误区

- **认为 Deal Flow 越多越好：** 数量不等于质量，过多的低质量项目会消耗团队精力，应追求"高质量的漏斗顶部"而非"尽可能多的项目"
- **忽视行业研究驱动的 Sourcing：** 很多基金过于依赖 FA 和被动接收，缺乏主动行业研究能力，导致只能看到"市场上的项目"而非"应该存在的项目"
- **筛选标准不一致：** 不同投资人员使用不同的筛选标准会导致资源浪费和团队决策分歧，应建立统一的评分卡（Scorecard）并定期校准
