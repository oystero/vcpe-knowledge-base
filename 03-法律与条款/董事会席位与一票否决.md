# 董事会席位与一票否决权

## 概述
董事会席位分配和一票否决权（Protective Provisions）是 VC/PE 投资中公司治理的核心安排，决定了投资人在企业重大决策中的话语权和保护能力。合理的治理结构既能保护投资人的合法权益，又不至于过度束缚创始团队的经营灵活性。在中国市场，随着公司治理意识的提升，董事会条款已成为机构投资人的标配要求。

## 核心内容

### 董事会席位分配

**典型董事会构成：**

| 席位类型 | 分配方 | 典型比例 | 说明 |
|----------|--------|----------|------|
| 创始人席位 | 创始团队 | 2-3席 | 包括CEO/CTO等核心管理层 |
| 投资人席位 | 领投方 | 1-2席 | 通常为领投方委派 |
| 独立董事 | 双方协商 | 1-2席 | 独立于各方的第三方 |
| 观察员席位 | 跟投方 | 1-2席 | 列席会议但无投票权 |

**不同阶段的董事会结构：**

| 阶段 | 董事会规模 | 创始人 | 投资人 | 独立 |
|------|-----------|--------|--------|------|
| 种子/天使 | 3-5人 | 2-3席 | 0-1席 | 0-1席 |
| A轮 | 5人 | 2-3席 | 1-2席 | 1席 |
| B轮 | 5-7人 | 2-3席 | 2-3席 | 1-2席 |
| C轮及以后 | 7-9人 | 2-3席 | 2-3席 | 2-3席 |
| Pre-IPO | 7-11人 | 视情况 | 视情况 | 独立董事占1/3+ |

**创始人控制 vs 投资人保护的平衡：**
- 创始人应保持多数席位（至少在早期阶段），确保经营决策的灵活性
- 投资人席位主要用于获取信息、参与重大决策，而非日常管理
- 独立董事可以在创始人与投资人僵持时起到调和作用

### 保护性条款（Protective Provisions / 一票否决权）

**核心概念：** 保护性条款赋予优先股持有人（通常需多数同意）对特定重大事项的否决权。即使董事会或股东会投票通过，若优先股持有人不同意，该事项无法执行。

**常见的保护性条款清单：**

| 类别 | 需否决的事项 | 说明 |
|------|-------------|------|
| **股权相关** | 发行新股或可转换证券 | 防止过度稀释 |
| | 创设新的优先股类别或系列 | 防止权利被劣后 |
| | 修改公司章程中影响优先股的条款 | 保护既有权利 |
| | 股票回购（员工回购除外） | 防止资产流失 |
| **债务相关** | 借债超过约定上限 | 控制财务风险 |
| | 为第三方提供担保 | 防止或有负债 |
| **资产相关** | 出售、转让或处置重大资产 | 防止核心资产流失 |
| | 进行并购或被并购 | 控制退出路径 |
| | 关联交易超过约定金额 | 防止利益输送 |
| **治理相关** | 变更审计师或会计政策 | 确保财务透明 |
| | 董事会规模或组成变更 | 维护治理平衡 |
| | 清算、解散或停业 | 保护退出权利 |
| **经营相关** | 变更主营业务方向 | 防止偏离投资主题 |
| | 超过约定金额的单笔投资或支出 | 控制资本使用 |

### 信息权（Information Rights）

| 信息类型 | 频率 | 内容 |
|----------|------|------|
| 财务报表 | 月度/季度/年度 | 资产负债表、利润表、现金流量表 |
| 经营数据 | 月度 | 关键KPI（营收、用户数、留存率等） |
| 预算与计划 | 年度 | 年度经营计划和预算 |
| 重大事项 | 即时 | 重大诉讼、关键人员变动、客户流失等 |
| 审计报告 | 年度 | 经审计的年度财务报告 |

### 观察员席位（Observer Seat）

**特点：**
- 列席董事会会议，获取全部会议材料
- 可发言讨论，但**无投票权**
- 通常分配给跟投方或次要投资人
- 成本低于正式董事席位，但信息获取等同

**实务考量：**
- 观察员通常需签署保密协议
- 某些情况下观察员可升级为正式董事（如领投方退出）
- 观察员是否受信义义务约束存在争议

### 中国市场的特殊实践

| 实践特点 | 说明 |
|----------|------|
| 有限公司 vs 股份公司 | 有限公司治理更灵活，股份公司受公司法更严格约束 |
| 监事会制度 | 中国公司法要求设监事会，与董事会形成双层治理 |
| 外商投资限制 | 部分行业对外资股东的董事席位有限制 |
| 政府引导基金 | 政府LP可能要求观察员席位或特定否决权 |
| 人民币基金偏好 | 人民币基金通常要求更强的治理参与权 |

### 一票否决权的谈判策略

**投资人侧（争取更多保护）：**
1. 列举详细的否决事项清单，覆盖所有重大风险
2. 要求降低否决的触发门槛（如"超过500万元"改为"超过100万元"）
3. 争取"负面承诺"（Negative Covenants），限制企业不得从事的行为

**创始人侧（保护灵活性）：**
1. 限制否决权的范围，仅限于核心事项
2. 设定金额门槛（如"超过年度预算20%的支出"才需批准）
3. 争取"日落条款"（Sunset Clause），否决权在特定条件后自动失效
4. 要求否决权持有人在约定时间内回应，避免"消极否决"

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 保护性条款 | Protective Provisions | 赋予优先股持有人对特定事项否决权的条款 |
| 董事会席位 | Board Seat | 在公司董事会中的代表权和投票权 |
| 观察员 | Observer | 列席董事会但无投票权的人员 |
| 独立董事 | Independent Director | 与公司无重大利益关系的外部董事 |
| 信息权 | Information Rights | 投资人获取公司经营和财务信息的权利 |
| 负面承诺 | Negative Covenants | 限制公司不得从事特定行为的承诺 |
| 日落条款 | Sunset Clause | 否决权在特定条件或时间后自动失效的条款 |
| 信义义务 | Fiduciary Duty | 董事对公司和股东负有的忠实义务和注意义务 |

## 实务要点
- **董事会构成反映权力格局**：创始人应确保在早期保持多数席位，避免过早让渡控制权
- **保护性条款要"精而不滥"**：过多的否决事项会拖慢决策效率，投资人应聚焦于真正需要保护的核心事项
- **定期审视治理结构**：每轮融资后重新评估董事会构成和否决权清单，确保与时俱进
- **独立董事的选择很重要**：独立董事应具有行业经验且各方信任，而非仅充数
- **信息权是"软保护"**：即使没有否决权，信息权也能让投资人及时发现问题

## 常见误区
- **将否决权等同于控制权**：否决权是"防守性"权利，只能阻止不能推动，不能用于强制公司采取行动
- **忽视否决权的行使成本**：频繁行使否决权会损害投资人与创始人的关系，降低公司运营效率
- **观察员等于董事**：观察员无投票权、无法对决议负责，在法律地位上与董事有本质区别
