# 优先清算权（Liquidation Preference）

## 概述
优先清算权（Liquidation Preference）是 VC/PE 优先股条款中最核心的经济权利之一，它决定了在公司发生清算事件（包括出售、合并、清算等）时，投资人相对于普通股股东的分配优先顺序和金额。优先清算权的设计直接影响投资人与创始人的退出利益分配，是估值谈判中"条款博弈"的核心战场。

## 核心内容

### 优先清算权的基本结构

**优先清算权 = 优先倍数 + 参与权类型**

| 要素 | 选项 | 说明 |
|------|------|------|
| 优先倍数（Preference Multiple） | 1x / 1.5x / 2x / 3x | 投资人优先回收的金额为投资额的倍数 |
| 参与权类型（Participation） | 参与型 / 非参与型 / 有上限参与型 | 确定优先回收后是否还参与剩余分配 |
| 清算事件定义 | 广义 / 狭义 | 哪些事件触发优先清算权 |

### 优先倍数

| 倍数 | 含义 | 市场分布 | 说明 |
|------|------|----------|------|
| 1x | 优先回收1倍投资额 | 最常见（80%+的交易） | 标准条款 |
| 1.5x-2x | 优先回收1.5-2倍投资额 | 少数交易 | 高风险项目或强势投资人 |
| 3x | 优先回收3倍投资额 | 极少见 | 困境融资或特殊情况 |

**1x 的含义：** 投资人投入 1000万元，在退出时至少优先获得 1000万元，之后剩余部分再按约定分配。

### 参与权类型（Participation Structure）

#### 非参与型优先股（Non-Participating Preferred）

**机制：**
- 投资人在退出时二选一：
  - 选项A：取回优先清算额（投资额 × 倍数）
  - 选项B：放弃优先清算权，按完全稀释后的持股比例参与分配
- 取两者中较高者

**示例（1x Non-Participating）：**
```
投资金额：1000万元，占股20%
退出场景A：公司以5000万元出售
  优先清算额 = 1000万元
  按比例分配 = 5000万 × 20% = 1000万元
  投资人获得 = max(1000万, 1000万) = 1000万元

退出场景B：公司以2亿元出售
  优先清算额 = 1000万元
  按比例分配 = 2亿 × 20% = 4000万元
  投资人获得 = max(1000万, 4000万) = 4000万元（选择按比例分配）
```

#### 参与型优先股（Participating Preferred / "Double Dip"）

**机制：**
- 投资人先取回优先清算额，**再**按持股比例参与剩余分配
- 相当于"双重分配"，对投资人有利，对创始人不利

**示例（1x Participating）：**
```
投资金额：1000万元，占股20%
退出价格：5000万元

第一步：投资人取回优先清算额 = 1000万元
第二步：剩余 = 5000万 - 1000万 = 4000万元
第三步：投资人参与剩余分配 = 4000万 × 20% = 800万元
投资人总获得 = 1000万 + 800万 = 1800万元（占总退出的36%）
```

#### 有上限参与型优先股（Participating Preferred with Cap）

**机制：**
- 参与型优先股，但总回报设有上限（通常 2x-3x）
- 达到上限后不再参与剩余分配

**示例（1x Participating, 3x Cap）：**
```
投资金额：1000万元
上限 = 3000万元
退出价格：2亿元

无上限时获得：1000万 + (2亿-1000万)×20% = 4800万元
有上限时获得：min(4800万, 3000万) = 3000万元
```

### 清算事件定义（Liquidation Event）

| 定义范围 | 涵盖事件 | 影响 |
|----------|----------|------|
| 广义 | 出售、合并、清算、资产转让、控制权变更 | 更有利于投资人 |
| 狭义 | 仅公司解散清算 | 更有利于创始人 |

**常见的广义清算事件：**
- 公司被收购（股权或资产）
- 公司与其他实体合并
- 出售全部或实质性全部资产
- 公司解散清算
- 控制权变更（Change of Control）

**创始人应注意：** 广义清算事件意味着任何退出都触发优先清算权，即使是"成功退出"（如被大公司收购）。

### 瀑布分配（Distribution Waterfall）

**标准瀑布顺序：**

```
退出总额
  ├─ 第一层：偿还投资人优先清算额（1x投资额）
  ├─ 第二层：普通股按比例分配剩余（如有参与权，投资人也参与）
  └─ 特殊：多轮优先股时，按"后进先出"（LIFO）原则
```

**多轮优先股的瀑布顺序：**

| 顺序 | 轮次 | 优先清算额 | 说明 |
|------|------|-----------|------|
| 1 | C轮（最后进入） | 1x 投资额 | 后进先出 |
| 2 | B轮 | 1x 投资额 | |
| 3 | A轮 | 1x 投资额 | 最先参与 |
| 4 | 剩余分配 | 按完全稀释比例 | 所有股东 |

**瀑布模拟示例（3轮投资，退出价5亿元）：**

| 轮次 | 投资额 | 占股 | 优先清算额 | 参与分配 | 总获得 |
|------|--------|------|-----------|---------|--------|
| C轮 | 2亿 | 25% | 2亿 | (5亿-4亿)×25%=2500万 | 2.25亿 |
| B轮 | 1.5亿 | 20% | 1.5亿 | 1亿×20%=2000万 | 1.7亿 |
| A轮 | 5000万 | 15% | 5000万 | 1亿×15%=1500万 | 6500万 |
| 普通股 | — | 40% | — | 1亿×40%=4000万 | 4000万 |

### 优先清算权的市场实践

**中国市场：**

| 优先级 | 中国市场常见做法 |
|--------|----------------|
| 1x Non-Participating | 最常见，约60%-70%的人民币基金交易 |
| 1x Participating | 约20%-30%，多见于PE项目 |
| 1x Participating with Cap | 约5%-10%，较温和的参与型 |
| 超过1x倍数 | 少见，多为特殊情况 |

**美元基金：**
- 1x Non-Participating 仍为主流（约65%）
- Participating Preferred 在后期轮次更常见
- 2x-3x 倍数在困境融资中出现

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 优先清算权 | Liquidation Preference | 优先股在退出时优先于普通股获得分配的权利 |
| 非参与型优先股 | Non-Participating Preferred | 只能在优先清算额和按比例分配中二选一 |
| 参与型优先股 | Participating Preferred | 先取回优先清算额，再按比例参与剩余分配 |
| 清算事件 | Liquidation Event | 触发优先清算权分配的事件 |
| 瀑布分配 | Distribution Waterfall | 退出收益在各层级投资者之间的分配顺序 |
| 双重分配 | Double Dip | 参与型优先股先取回投资再参与分配的机制 |
| 后进先出 | Last In, First Out (LIFO) | 多轮投资中最后进入的投资人最先获得优先分配 |

## 实务要点
- **参与型 vs 非参与型的差距巨大**：在退出价为投资2-4倍时，参与型比非参与型多获得30%-100%的回报
- **退出价越高，优先清算权越不重要**：当退出价远高于投资额时，所有人都按比例分配，优先清算权变得无关紧要
- **多轮投资的瀑布顺序要清晰**：不同轮次的优先权顺序（Pari Passu vs LIFO）会导致巨大的分配差异
- **"清算事件"的定义是谈判重点**：创始人应争取狭义定义，排除并购退出；投资人应争取广义定义
- **模拟不同退出场景**：在签署条款前，必须模拟退出价在1x-10x之间的分配情况，确保各方理解实际影响

## 常见误区
- **低估参与型优先股的影响**：创始人常常不了解 Participating Preferred 的"双重分配"本质，导致在谈判中过度让步
- **忽略优先清算权的"保底"功能**：对于投资人而言，1x Non-Participating Preferred 是最基本的下行保护，不应被视为"苛刻条款"
- **多轮优先权的叠加效应**：3轮各1x的优先清算总额等于3倍投资额，中等退出价下普通股可能所剩无几
