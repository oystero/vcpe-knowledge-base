# 反稀释条款（Anti-Dilution Protection）

## 概述
反稀释条款（Anti-Dilution Clause）是 VC/PE 投资中优先股的核心保护条款之一，旨在保护投资人在后续融资轮次估值下调（Down Round）时免受过度稀释。当企业以低于前一轮的价格发行新股时，反稀释条款会调整前轮投资人的转换价格，使其获得更多普通股，从而维持其持股比例和经济利益。

## 核心内容

### 反稀释条款的两种主要类型

#### 1. Full Ratchet（完全棘轮）

**机制：**
- 转换价格直接调整为后续轮次的发行价格
- 无论后续轮次的发行数量多少，均以最低价格为准

**公式：**
```
新转换价格 = 后续轮次的每股价格
```

**示例：**
| 参数 | A轮 | B轮（Down Round） |
|------|------|------|
| 每股价格 | 10元 | 5元 |
| A轮投资人持有 | 100万股优先股 | — |
| 转换价格（Full Ratchet） | 10元 | 调整为5元 |
| 可转换普通股数 | 100万股 | 调整为200万股 |

**特点：** 对投资人保护最强，但对创始人和其他股东的稀释影响极大。

#### 2. Weighted Average（加权平均）

**机制：**
- 考虑后续轮次的发行规模，按加权平均方式调整转换价格
- 区分为"广义加权平均"（Broad-Based）和"狭义加权平均"（Narrow-Based）

**公式：**
```
新转换价格 = 原转换价格 × (已发行股数 + 按原价可购新股数) / (已发行股数 + 实际发行新股数)

即：CP₂ = CP₁ × (CS₁ + CS₁') / (CS₁ + CS₂)

其中：
CP₁ = 原转换价格
CS₁ = 下轮前完全稀释总股数
CS₁' = 本轮融资额按原价可购股数 = 本轮融资额 / CP₁
CS₂ = 本轮实际发行股数 = 本轮融资额 / 下轮每股价格
```

**Broad-Based vs Narrow-Based：**

| 类型 | 已发行股数（CS₁）的口径 | 对投资人的保护 |
|------|------------------------|---------------|
| Broad-Based（广义） | 包含全部已发行股份、期权池、可转换证券 | 较弱（稀释被分摊到更大基数） |
| Narrow-Based（狭义） | 仅包含已发行优先股 | 较强（稀释仅在优先股之间分摊） |

### 计算示例

**场景设定：**
- A轮：投资1000万元，每股10元，获得100万股优先股
- A轮前完全稀释总股本：900万股（含期权池）
- B轮：融资2000万元，每股5元（Down Round）

**Full Ratchet 计算：**
```
新转换价格 = 5元
A轮可转换普通股 = 1000万 / 5 = 200万股
新增稀释 = 200万 - 100万 = 100万股额外获得
```

**Broad-Based Weighted Average 计算：**
```
CS₁ = 900万股（完全稀释总股本）
CS₁' = 1000万 / 10 = 100万股
CS₂ = 2000万 / 5 = 400万股

新转换价格 = 10 × (900 + 100) / (900 + 400)
           = 10 × 1000 / 1300
           = 7.69元

A轮可转换普通股 = 1000万 / 7.69 = 130万股
新增稀释 = 130万 - 100万 = 30万股额外获得
```

**对比总结：**

| 方式 | 调整后转换价格 | A轮可转换股数 | 额外获得 |
|------|-------------|-------------|---------|
| 无反稀释 | 10元 | 100万股 | 0 |
| Broad-Based WA | 7.69元 | 130万股 | 30万股 |
| Narrow-Based WA | 6.25元 | 160万股 | 60万股 |
| Full Ratchet | 5元 | 200万股 | 100万股 |

### Pay-to-Play 条款

**机制：**
- 要求投资人在后续轮次中继续跟投，否则失去反稀释保护
- 将优先股降级为普通股或"死优先股"（Zombie Preferred）

**目的：**
- 确保投资人在企业困难时继续提供资金支持
- 防止投资人"搭便车"——享受保护但不承担义务

**Pay-to-Play 的典型设计：**

| 设计要素 | 选项 | 说明 |
|----------|------|------|
| 要求参与程度 | 必须参与后续轮次 | 100%参与或按比例参与 |
| 未参与后果 | 丧失反稀释保护 | 降级为普通股或次级优先股 |
| 宽限期 | 允许一定时间窗口 | 通常30-90天 |
| 适用轮次 | 仅限Down Round | 或所有后续轮次 |

### 反稀释的例外与排除

以下情况通常不触发反稀释调整：
- 员工期权池的发行（ESOP）
- 债务转换为股权（除非以低于转换价发行）
- 股票分红或拆股
- 并购中发行的对价股份
- 经优先股多数持有人同意的发行

### 中国市场的特殊实践

| 实践特点 | 说明 |
|----------|------|
| 以Weighted Average为主 | Full Ratchet 在中国市场相对少见 |
| Pay-to-Play较少 | 中国LP/GP关系与美国不同，条款较少使用 |
| 反稀释与对赌叠加 | 部分交易同时设置反稀释和业绩对赌 |
| 人民币基金条款较温和 | 相比美元基金，人民币基金的保护条款通常较弱 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 完全棘轮 | Full Ratchet | 转换价格直接调整为后续最低发行价格的反稀释方式 |
| 加权平均 | Weighted Average | 按发行规模加权调整转换价格的反稀释方式 |
| 广义加权平均 | Broad-Based Weighted Average | 以全部稀释后股本为基础计算的加权平均 |
| 狭义加权平均 | Narrow-Based Weighted Average | 仅以已发行优先股为基础计算的加权平均 |
| 转换价格 | Conversion Price | 优先股转换为普通股的价格 |
| 估值下调融资 | Down Round | 后续融资轮次估值低于前一轮的融资 |
| 死优先股 | Zombie Preferred | 失去反稀释等保护后仅保留基本权利的优先股 |
| 跟投义务 | Pay-to-Play | 要求投资人在后续轮次继续投资的义务 |

## 实务要点
- **Broad-Based Weighted Average 是市场主流**：绝大多数美元基金的标准化条款采用此方式，中国市场也以此为主
- **创始人应争取排除 Full Ratchet**：Full Ratchet 对创始人极其不利，一轮 Down Round 可能导致创始人丧失控股权
- **注意反稀释与优先清算权的叠加效应**：反稀释增加持股比例 + 优先清算权保障退出，两者叠加对普通股影响巨大
- **计算 Cap Table 影响**：任何反稀释条款都应在 Cap Table 模型中模拟其影响，看清 Down Round 各方持股变化
- **Pay-to-Play 是双刃剑**：对投资人而言可能限制灵活性，但有助于筛选出真正支持企业的投资人

## 常见误区
- **认为反稀释只是"技术条款"**：反稀释在 Down Round 中可能导致创始人持股比例大幅下降，绝非无关紧要的细节
- **混淆 Broad-Based 和 Narrow-Based**：两者计算结果差异显著（在上述示例中相差50%），必须在条款中明确
- **忽视排除条款**：期权池发行、并购对价等通常被排除在反稀释触发之外，需确认排除清单是否完整
