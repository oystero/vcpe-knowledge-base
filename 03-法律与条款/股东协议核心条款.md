# 股东协议核心条款

## 概述
股东协议（Shareholders' Agreement, SHA）是规范公司各股东之间权利义务关系的核心法律文件，在 VC/PE 投资中与投资协议、公司章程共同构成交易的法律框架。股东协议中的核心条款——优先购买权、共同出售权、拖带权、锁定期和知情权——直接关系到投资人的退出渠道、信息获取和股权流动性，是每一位投资人和创始人都必须深入理解的基本条款。

## 核心内容

### 优先购买权（Right of First Refusal, ROFR）

**机制：**
- 现有股东拟向第三方转让股份时，其他股东有权在同等条件下优先购买
- 目的是维护现有股东的持股比例，防止引入不受欢迎的新股东

**关键要素：**

| 要素 | 内容 | 谈判要点 |
|------|------|----------|
| 适用范围 | 所有股份转让 或 仅特定类别的股份 | 投资人通常要求覆盖所有股份 |
| 通知要求 | 转让方需书面通知拟转让的数量和价格 | 通知期限通常30-60天 |
| 行使期限 | 收到通知后的回应时间 | 通常15-30天 |
| 等同条件 | 其他股东需按同等价格和条件购买 | 包括价格、付款方式、交割条件 |
| 例外情况 | 员工期权、关联方转让、遗产继承等 | 通常被豁免 |

**与其他条款的配合：**
- ROFR + 共同出售权 → 双重保护
- ROFR + 拖带权 → 覆盖主动退出和被动退出

### 共同出售权（Tag-Along Right / Co-Sale Right）

**机制：**
- 当大股东（通常是创始人）向第三方出售股份时，小股东（通常是投资人）有权按比例参与出售
- 保护小股东在大股东"套现"时也能获得流动性

**计算示例：**
```
创始人持有：60% 股份
投资人持有：30% 股份
其他股东：10% 股份

创始人拟向第三方出售 20% 股份（即其持股的1/3）

投资人共同出售权：
  可出售比例 = 30% / 60% × 20% = 10%（占总股本）
  即投资人可按同比例参与出售

第三方实际获得：
  20%（创始人）+ 10%（投资人共同出售）= 30%
```

**关键谈判要点：**

| 议题 | 投资人立场 | 创始人立场 |
|------|-----------|-----------|
| 是否设置最低出售门槛 | 不设门槛，任何出售都可参与 | 设定门槛（如超过5%才触发） |
| 价格是否需一致 | 必须按同一价格 | 可接受分批定价 |
| 是否可超额行使 | 不可超过比例 | 不可超过比例 |
| 通知和行使期限 | 尽量短（10-15天） | 尽量长（30天+） |

### 拖带权（Drag-Along Right）

**机制：**
- 当特定比例的股东同意出售公司时，可强制要求其他股东按同一条件参与出售
- 确保少数股东不会阻挠公司的整体出售

**触发条件：**

| 条件类型 | 常见设计 | 说明 |
|----------|----------|------|
| 股东比例 | 50%-75% 的股东同意 | 需多数或绝对多数同意 |
| 估值门槛 | 不低于约定估值 | 保护小股东不在低价时被迫出售 |
| 时间限制 | 投资后N年才可行使 | 避免过早退出 |
| 董事会批准 | 需董事会决议通过 | 增加一层审查 |

**拖带权的博弈：**
- **投资人立场**：希望拥有拖带权，以在合适时机推动退出，不受少数股东阻挠
- **创始人立场**：担心在估值不理想时被强制出售，通常要求设定估值下限
- **少数股东立场**：拖带权是对其权益的最大威胁，通常要求保护性条款

**拖带权 vs 共同出售权对比：**

| 维度 | 拖带权（Drag-Along） | 共同出售权（Tag-Along） |
|------|---------------------|----------------------|
| 触发方 | 多数股东 | 少数股东 |
| 方向 | 强制跟卖 | 自愿跟卖 |
| 保护对象 | 保护多数股东的退出权 | 保护少数股东的退出权 |
| 行使方式 | 强制性 | 选择性 |
| 估值影响 | 通常有利（整体出售溢价） | 取决于出售价格 |

### 锁定期（Lock-Up Period）

**机制：**
- 在特定期间内限制股东转让股份
- 常见于 IPO 前后的股份锁定安排

**不同类型：**

| 类型 | 期间 | 目的 |
|------|------|------|
| 投资后锁定期 | 投资完成后1-3年 | 确保创始人持续投入 |
| IPO 锁定期 | IPO后6-12个月 | 防止上市后集中抛售（证券监管要求） |
| 竞业锁定期 | 在职及离职后1-2年 | 与竞业禁止义务配合 |
| 分批解锁 | 4年期限，1年cliff | 类似期权归属（Vesting） |

**Vesting 条款（创始人股份归属）：**
```
典型设计：
- 总期限：4年
- Cliff（悬崖期）：1年（满1年才解锁25%）
- 之后：按月或按季度线性解锁
- 加速归属（Acceleration）：
  - 单触发（Single Trigger）：公司被收购时立即全部归属
  - 双触发（Double Trigger）：收购+被解雇才加速归属
```

### 知情权（Information Rights）

**机制：**
- 投资人有权获取公司的经营和财务信息
- 是保护性条款的基础——没有信息就无法判断是否需要行使否决权

**知情权的具体内容：**

| 信息类别 | 频率 | 详细程度 |
|----------|------|----------|
| 年度审计报告 | 年度 | 经审计的完整财务报表 |
| 季度管理报表 | 季度 | 资产负债表、利润表、现金流 |
| 月度经营数据 | 月度 | 收入、成本、关键KPI |
| 年度预算 | 年度 | 下一年度经营计划和财务预算 |
| 重大事项通知 | 即时 | 重大诉讼、人事变动、客户变故、政策影响等 |
| 董事会材料 | 每次会议前 | 议程、决议草案、相关材料 |

**中国市场的知情权实践：**
- 有限公司：股东有权查阅公司章程、股东会决议、财务报告（《公司法》第33条）
- 股份公司：股东有权查阅公司章程、股东名册、财务报告等（《公司法》第97条）
- PE投资人通常要求更广泛的知情权，超越法定范围

### 股东协议中的其他重要条款

| 条款 | 说明 |
|------|------|
| 优先认购权（Pre-emptive Right） | 公司发行新股时，现有股东有权按比例认购以维持持股比例 |
| 反稀释条款 | 前文详述，保护投资人在Down Round时免受过度稀释 |
| 优先清算权 | 前文详述，决定退出时的分配优先级 |
| 竞业禁止与保密 | 限制创始人在投资期间和之后的竞争行为 |
| 分红政策 | 约定利润分配的比例和频率 |
| 争议解决条款 | 仲裁还是诉讼，适用哪个法律 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 优先购买权 | Right of First Refusal (ROFR) | 股份转让时其他股东优先购买的权利 |
| 共同出售权 | Tag-Along Right / Co-Sale Right | 大股东出售时小股东按比例参与出售的权利 |
| 拖带权 | Drag-Along Right | 多数股东强制少数股东参与出售的权利 |
| 锁定期 | Lock-Up Period | 限制股份转让的期间 |
| 股份归属 | Vesting | 创始人股份按时间或里程碑逐步获得完全所有权 |
| 加速归属 | Acceleration | 特定事件触发时提前完成股份归属 |
| 知情权 | Information Rights | 股东获取公司经营和财务信息的权利 |
| 优先认购权 | Pre-emptive Right / Pro-rata Right | 新股发行时按比例认购的权利 |
| 竞业禁止 | Non-Compete | 限制股东从事竞争性业务的义务 |

## 实务要点
- **ROFR 和 Tag-Along 通常同时存在**：ROFR 保护现有股东不被"外来者"稀释，Tag-Along 保护少数股东不在大股东退出时被"抛弃"
- **Drag-Along 要设定保护门槛**：无条件的拖带权可能被滥用，应设定估值下限、时间窗口和董事会批准要求
- **Vesting 条款保护所有人**：不仅保护投资人（防止创始人早期离职），也保护创始人（展示承诺获得投资人信任）
- **知情权是"沉默的保护者"**：很多时候投资人并不需要行使否决权，只需通过知情权了解情况，就能及时预警
- **股东协议与公司章程需一致**：在中国法律下，公司章程具有最高效力，股东协议中的条款如需对抗第三方，应纳入章程

## 常见误区
- **将拖带权视为"恶意条款"**：拖带权对所有股东都有利——它确保公司在合适价格退出时不会被少数股东阻挠
- **忽视 Vesting 的重要性**：创始人不设 Vesting 可能导致早期离职仍保留大量股份，对其他股东不公平
- **混淆 ROFR 与 Pre-emptive Right**：前者针对二级市场转让，后者针对公司发行新股；两者触发场景不同
- **知情权条款过于笼统**：缺乏具体的信息类型、频率和格式约定，导致执行时产生争议
