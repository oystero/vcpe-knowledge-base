# 对赌协议（VAM）

## 概述
对赌协议，正式名称为估值调整机制（Valuation Adjustment Mechanism, VAM），是 VC/PE 投资中用于弥合投资人与融资方之间估值分歧的法律安排。它通过对未来不确定事项设定条件，约定在条件达成或未达时对投资价格进行调整。在中国 PE/VC 实践中，对赌协议是最具争议也最广泛使用的投资条款之一，其司法效力经历了从"全面否定"到"有条件认可"的演变。

## 核心内容

### 对赌协议的主要类型

| 类型 | 触发条件 | 补偿方式 | 常见场景 |
|------|----------|----------|----------|
| 业绩对赌（盈利补偿） | 净利润/营收未达承诺 | 股份补偿或现金补偿 | PE 投资成熟企业 |
| 上市对赌 | 未在约定期限内完成IPO | 回购股份（投资额+利息） | Pre-IPO 投资 |
| 里程碑对赌 | 产品获批/收入突破等 | 估值调整或追加投资 | 医药、科技企业 |
| 反向对赌 | 业绩超预期 | 投资人让渡部分股权 | 激励型安排 |
| 综合对赌 | 多重条件组合 | 多种补偿方式组合 | 复杂交易 |

### 触发条件设计

**业绩指标选择：**

| 指标 | 优势 | 劣势 |
|------|------|------|
| 净利润 | 直观反映盈利能力 | 可通过会计手段调节 |
| 营收 | 不易操纵 | 不反映盈利能力 |
| EBITDA | 排除资本结构影响 | 计算口径可能有争议 |
| 经营性现金流 | 最难操纵 | 波动性大 |
| 非财务指标 | 产品获批、用户数等 | 难以量化补偿金额 |

**关键设计要素：**
- 承诺期间：通常 2-4 年，与基金存续期匹配
- 承诺金额：基于管理层预测的 80%-90% 设定（留有安全边际）
- 考核频率：通常按年度考核，部分按季度
- 补偿上限：通常不超过投资金额的 100%-150%

### 补偿机制计算

**1. 现金补偿模式：**
```
补偿金额 = 投资额 × (1 - 实际利润/承诺利润)
```

**2. 股份补偿模式：**
```
应回购股份数 = 投资额 × (1 - 实际利润/承诺利润) / 回购价格
回购价格 = 投资额 / 原获得股份数 或 约定价格
```

**3. 回购模式（上市失败触发）：**
```
回购金额 = 投资额 × (1 + 回购利率)^n
回购利率：通常 8%-15%（单利或复利）
```

### 中国司法实践：里程碑式判例

#### 海富案（最高人民法院，(2012)民提字第11号）

**案件背景：**
- 2007年，苏州海富投资以2000万元增资甘肃世恒（一家生产型企业）
- 对赌条款：若2008年净利润低于3000万元，世恒需补偿海富
- 实际净利润：约2.7万元

**判决结果与规则：**
- 投资人与**目标公司**的对赌条款 → **无效**（损害公司及债权人利益）
- 投资人与**股东/实际控制人**的对赌条款 → **有效**（当事人真实意思表示）

**确立的规则：**
> 投资方与目标公司的对赌协议因损害公司及债权人利益而无效；投资方与目标公司股东之间的对赌协议，如无其他无效事由，应认定有效。

#### 华工案（江苏省高级人民法院，(2019)苏民再62号）

**案件背景：**
- 华工创投投资扬州锻压机床公司
- 约定目标公司若未在规定时间内上市，需回购股份
- 二审认定回购条款有效，但目标公司主张无效

**判决结果与规则：**
- 在满足**资本维持原则**和**不损害债权人利益**的前提下，目标公司的回购义务可以有效
- 但需审查是否履行了法定的**减资程序**
- 实际执行需配合公司法下的减资决议程序

**华工案的进步意义：**
- 放宽了海富案的严格限制
- 引入了"资本维持原则审查+减资程序"的判断标准
- 但仍要求实质审查，非一概认可

#### 2019年《九民纪要》的最新规则

**最高人民法院《全国法院民商事审判工作会议纪要》（2019年11月）：**

| 情形 | 效力 | 说明 |
|------|------|------|
| 与目标公司对赌（回购） | 原则有效，但需审查减资程序 | 未完成减资程序的，法院不支持实际履行 |
| 与目标公司对赌（金钱补偿） | 原则有效，但需审查是否有利润可分配 | 无利润则不支持补偿 |
| 与股东/实控人对赌 | 有效 | 原则不变 |

### 对赌条款的实务要点

**投资人视角的保护设计：**
1. 优先与创始人/实控人签订对赌，而非目标公司
2. 对赌指标应多维度（净利润+营收+非财务指标）
3. 设置补偿上限和下限，避免过度惩罚
4. 约定争议解决条款（仲裁优于诉讼）

**创始人视角的风险防控：**
1. 承诺金额应保守，留有安全边际（建议取管理层预测的70%-80%）
2. 要求设置"上限条款"（Cap），限制最大补偿金额
3. 争取"阶梯式补偿"（而非线性补偿），减少微小偏差的惩罚
4. 避免"无限连带责任"，将对赌义务限定在特定资产范围内

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 估值调整机制 | Valuation Adjustment Mechanism (VAM) | 基于未来不确定事项对投资估值进行事后调整的合约安排 |
| 业绩承诺 | Performance Commitment / Profit Guarantee | 融资方对未来财务业绩的承诺 |
| 股份补偿 | Share Compensation | 以退还或调整股份的方式进行估值补偿 |
| 现金补偿 | Cash Compensation | 以现金方式进行估值补偿 |
| 回购权 | Redemption Right / Put Option | 投资人在特定条件下要求融资方回购其股份的权利 |
| 资本维持原则 | Capital Maintenance Principle | 公司法要求公司维持与注册资本相当的资产，保护债权人利益 |
| 九民纪要 | Minutes of the National Courts' Civil and Commercial Trial Work Conference | 最高人民法院2019年发布的审判指导文件 |

## 实务要点
- **对赌的可执行性是核心**：条款写得再漂亮，如果目标公司无力补偿、创始人无力回购，就是一纸空文；尽调时需评估补偿方的实际偿付能力
- **中国法下对目标公司对赌要特别谨慎**：海富案和九民纪要的规则意味着，与目标公司的对赌虽然可能有效，但执行受减资程序制约
- **注意税务影响**：对赌补偿可能涉及所得税（视为股权转让所得或偶发所得），各地税务处理不统一
- **对赌期间的信息权**：应约定投资人对被投企业的知情权和检查权，以便监控业绩完成情况
- **仲裁条款的选择**：对赌争议建议约定仲裁而非诉讼，仲裁保密性强、效率高

## 常见误区
- **"与公司签对赌无效"**：海富案后很多人认为与目标公司的对赌一律无效，但九民纪要已修正为"原则有效，需审查减资/利润"
- **对赌只保护投资人**：反向对赌（业绩超预期时奖励创始人）是公平的对赌设计，应积极争取
- **忽视对赌的长期关系影响**：过度苛刻的对赌条款可能导致创始人行为扭曲（如短期利润导向、推迟研发投入），损害企业长期价值
