# 基金架构与法律实体

## 概述
基金架构决定了 VC/PE 基金的法律形式、税务处理、监管要求和投资者保护水平。主流架构包括有限合伙制（Limited Partnership）、公司制（Corporate）和契约制（Contractual），不同司法管辖区（开曼、香港、境内）的选择对基金运营有深远影响。

## 核心内容

### 三种主流基金架构对比

| 维度 | 有限合伙制 | 公司制 | 契约制 |
|------|-----------|--------|--------|
| 法律依据 | 合伙企业法 | 公司法 | 信托法/基金法 |
| 法律主体 | 合伙企业（非独立法人） | 独立法人 | 信托/资管计划 |
| 税务处理 | 穿透纳税（Pass-Through） | 双重征税（公司税+分红税） | 穿透或免税 |
| 决策机制 | GP 负责日常管理 | 董事会+股东会 | 管理人负责 |
| 责任承担 | GP 无限责任，LP 有限责任 | 股东以出资为限 | 信托隔离 |
| 灵活性 | 高（LPA 自由约定） | 中（受公司法限制） | 中 |
| 行业主流度 | **最主流**（全球 90%+） | 部分使用 | 部分境内基金 |
| 适用场景 | 国际 VC/PE | 上市基金、母基金 | 境内私募基金 |

### 有限合伙制详解（Limited Partnership）

有限合伙制是全球 VC/PE 行业最主流的基金架构。

**基本结构：**
```
                    ┌─────────────────┐
                    │  管理公司        │
                    │ (Management Co.) │
                    │  GP 持股 100%    │
                    └────────┬────────┘
                             │ 管理协议
                    ┌────────▼────────┐
                    │   基金主体       │
                    │ (Limited         │
                    │  Partnership)    │
                    │  GP + LP         │
                    └────────┬────────┘
                             │ 投资
                    ┌────────▼────────┐
                    │   被投企业       │
                    │ (Portfolio       │
                    │  Companies)      │
                    └─────────────────┘
```

**核心文件：**
- LPA（Limited Partnership Agreement）：有限合伙协议，基金的"宪法"
- Subscription Agreement：出资协议，LP 的出资承诺
- Management Agreement：管理协议，GP 与基金之间的管理安排
- Side Letter：附属协议，特定 LP 的个性化条款

### 开曼群岛架构（Cayman Islands）

开曼是全球最主流的离岸基金注册地。

**开曼基金类型：**

| 类型 | 适用场景 | 监管强度 |
|------|----------|---------|
| Exempted Limited Partnership（豁免有限合伙） | 最常用的 VC/PE 基金架构 | 低（需注册，无审计要求） |
| Segregated Portfolio Company（SPC） | 母基金、FoF | 中 |
| Registered Fund（注册基金） | 对冲基金、公开募集 | 高 |

**开曼架构的优势：**
- 零税率（无所得税、资本利得税、预提税）
- 灵活的 LPA 定制空间
- 全球 LP 普遍接受
- 成熟的法律体系（英国普通法）
- 保密性高

**开曼架构的劣势：**
- CRS/AEOI 信息交换要求增加合规成本
- 部分 LP（特别是欧洲养老金）对开曼架构有 ESG 顾虑
- 经济实质法（Economic Substance Act）增加了合规要求

### 香港架构

**香港有限合伙基金（LPF）：**
- 2020 年 8 月实施《有限合伙基金条例》
- 对标开曼 ELP，吸引基金在香港注册
- 税务优惠：符合条件的基金可享受利得税豁免
- 优势：靠近中国大陆市场、国际认可度高
- 劣势：设立成本高于开曼、法律体系仍在完善中

### 境内架构（中国大陆）

**三种境内基金形式：**

| 形式 | 法律依据 | 适用场景 | 特点 |
|------|----------|----------|------|
| 有限合伙型 | 《合伙企业法》 | 最主流的 PE/VC 基金 | 穿透纳税、GP 管理 |
| 公司型 | 《公司法》 | 部分政府引导基金 | 双重征税、治理规范 |
| 契约型 | 《证券投资基金法》 | 私募证券基金为主 | 信托结构、灵活 |

**境内基金的监管要求：**
- 私募基金管理人登记（基金业协会 AMAC）
- 基金产品备案
- 合格投资者认定
- 定期信息披露

### 典型跨境基金架构

大型美元基金通常采用多层架构以满足不同需求：

```
                    ┌───────────────┐
                    │  GP / 管理公司  │
                    │  (开曼/BVI)    │
                    └───────┬───────┘
                            │
           ┌────────────────┼────────────────┐
           │                │                │
    ┌──────▼──────┐  ┌──────▼──────┐  ┌──────▼──────┐
    │   美元基金    │  │   人民币基金  │  │  平行基金    │
    │   (开曼ELP)  │  │  (境内LP)   │  │  (开曼ELP)  │
    └──────┬──────┘  └──────┬──────┘  └──────┬──────┘
           │                │                │
    ┌──────▼──────┐  ┌──────▼──────┐        │
    │  红筹/VIE   │  │  境内直投    │  同上结构
    │   架构企业   │  │   企业      │
    └─────────────┘  └─────────────┘
```

### 人民币基金 vs 美元基金

| 维度 | 人民币基金 | 美元基金 |
|------|-----------|---------|
| 主要注册地 | 境内（有限合伙）/ 部分开曼 | 开曼 / 香港 |
| LP 构成 | 政府引导基金、险资、银行理财、FoF | 养老金、主权基金、大学捐赠基金、FoF |
| 币种 | 人民币 | 美元 |
| 退出渠道 | A 股 IPO、境内并购 | 港股/美股 IPO、海外并购 |
| 监管 | 基金业协会（AMAC）+ 发改委 | 开曼 CIMA / 香港 SFC |
| 税务 | 穿透纳税（20% 个人所得税） | 离岸免税 |
| 典型规模 | 5-50 亿人民币 | 1-10 亿美元 |

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 有限合伙企业 | Limited Partnership (LP) | GP 承担无限责任、LP 承担有限责任的合伙组织 |
| 豁免有限合伙 | Exempted Limited Partnership (ELP) | 开曼群岛最常见的基金法律形式 |
| 管理公司 | Management Company | GP 设立的基金管理实体，收取管理费 |
| 附属协议 | Side Letter | 为特定 LP 定制的个性化条款协议 |
| 红筹架构 | Red Chip Structure | 境内权益通过境外 SPV 持有的架构 |
| VIE 架构 | Variable Interest Entity | 通过协议控制实现境外上市的架构 |
| 合格投资者 | Qualified/Accredited Investor | 符合法定资产/收入标准的投资人 |

## 实务要点

- **架构选择要"以终为始"：** 基金架构应根据目标 LP、投资策略和退出路径来设计，而非简单套用模板
- **开曼 ELP 仍是美元基金的首选：** 尽管面临越来越多的合规要求，开曼 ELP 在税务效率、法律灵活性和全球认可度上仍具优势
- **人民币基金要注意"双 GP"结构：** 部分地方政府引导基金要求在基金中担任 GP 或 LP+Co-GP，需注意权利义务划分
- **跨境架构要关注外汇管制：** 人民币基金投海外或美元基金投境内，均需考虑外汇登记和资金出入境安排
- **Side Letter 谈判是 LP 的重要工具：** 大型 LP 应充分利用 Side Letter 争取 Co-Invest 权、Fee Discount、信息权等附加权益

## 常见误区

- **认为有限合伙制只是一种法律形式：** 实际上，有限合伙制的灵活性（LPA 自由约定）是 VC/PE 行业能够快速发展的关键制度基础，不同 LPA 条款的差异可能比不同法律形式的差异更大
- **忽视税务穿透的实际影响：** 穿透纳税意味着基金层面不缴税，利润直接穿透到 LP 个人/机构层面纳税，不同 LP 的税务身份（境内/境外、个人/机构）会导致税务处理的显著差异
- **盲目追求最复杂的架构：** 多层架构虽然有税务优化等好处，但也增加了设立成本、运营复杂度和合规负担，对于中小型基金来说简单架构可能更优
