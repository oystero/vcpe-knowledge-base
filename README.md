## 深度研报阅读指引

04-行业赛道目录下的 `深度研报/` 子目录包含 8 份行业深度研报，覆盖：

- SaaS与企业服务、人工智能、先进制造与机器人、半导体与硬科技
- 新能源与碳中和、消费与零售、生物科技与医疗、金融科技

**阅读建议：**

1. **先读概述，再读研报**：每份研报对应一篇行业概述文件。先读概述了解基本面，再通过研报深入产业链分析、竞争格局、估值逻辑。
2. **结合赛道案例**：研报中的理论框架可通过赛道案例验证。例如读完 [[人工智能-深度研报]]，再看 [[案例-赛道-DeepMind]] 和 [[案例-赛道-智谱AI]]。
3. **关注投资视角**：研报不是行业百科，重点看"投资机会在哪""风险点是什么""估值怎么定"三个问题。

## 案例库使用方法

10-案例库包含 106 个真实投资案例，按五个维度分类：

### 案例分类

| 类型 | 数量 | 说明 |
|------|------|------|
| 早期成功案例 | 20 | 字节跳动、美团、拼多多等中国独角兽的早期投资复盘 |
| 成长期成功案例 | 20 | 宁德时代、药明康德等企业从成长到上市的投资逻辑 |
| 失败/争议案例 | 20 | OFO、瑞幸、WeWork 等失败案例的教训总结 |
| 海外经典案例 | 20 | Facebook、Tesla、OpenAI 等全球标杆案例 |
| 行业专题案例 | 20 | Salesforce、台积电、比亚迪等行业龙头的赛道分析 |

### 案例使用方式

- **对标研究**：投资某赛道时，先看该赛道的专题案例，理解行业龙头的发展路径和投资逻辑。
- **风险警示**：失败案例是最好的老师。每个失败案例都标注了核心教训和尽调红旗信号。
- **估值参考**：成功案例中包含融资历程和估值变化，可作为可比交易的数据来源。
- **案例对比**：将同赛道的早期案例和成长期案例对比，观察企业从 0 到 1 再到 N 的完整路径。

## Web 应用

知识库附带 Astro 构建的 Web 应用，支持在线浏览、搜索和知识图谱可视化：

```bash
cd site

# 安装依赖
npm install

# 生成内容数据（从 Markdown 文件生成 JSON）
node scripts/fetch-content.mjs

# 启动开发服务器
npm run dev

# 构建静态站点
npm run build
```

### 功能特性

| 功能 | 说明 |
|------|------|
| 全文搜索 | Fuse.js 驱动，支持标题/内容/标签搜索，支持标签筛选 |
| 知识图谱 | D3.js 力导向图，展示文章间的双向链接关系，支持拖拽和缩放 |
| 模块导航 | 12 个知识模块的分类浏览 |
| 文章渲染 | Markdown → HTML，支持 Wiki 链接、表格、Callout |
| 目录导航 | 自动生成文章目录，滚动高亮当前位置 |
| 相关推荐 | 基于标签和链接的相关文章推荐 |
| 暗色主题 | 支持暗色/亮色切换，自动跟随系统偏好 |
| 响应式设计 | 移动端自适应布局 |

### 构建产物

- 183 个静态 HTML 页面
- 8.5 MB 总大小
- 可部署到任何静态托管服务（Netlify、Vercel、GitHub Pages 等）

## 推荐学习路径

### 路径一：零基础入门（2-3 周）

4. **案例入门**：[[案例-早期-字节跳动]] → [[案例-失败-OFO小黄车]]

### 路径二：估值能力进阶（2 周）

1. **估值基础**：[[Pre-Money与Post-Money]] → [[现金流折现法-DCF]] → [[可比公司法]]

2. **深入研报**：阅读对应深度研报（如 [[人工智能-深度研报]]）
